table: fact.ky035
description: Kontanthjælp (helårsmodtagere) efter område, visitation og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- visitation: values [TOT=I alt, 1=Jobparat, 2=Aktivitetsparat, 3=Åbenlyst uddannelsesparat, 4=Uddannelsesparat, 9=Uoplyst]
- tid: date range 2007-01-01 to 2024-01-01
dim docs: /dim/nuts.md
notes:
- This table has NO tal selector — indhold directly represents helårsmodtagere (annual FTE recipients). No persons count or payments available here.
- omrade joins dim.nuts on omrade=kode. 98% match. omrade=0=Hele landet, omrade=997=Udlandet not in dim.nuts.
- Use ColumnValues("nuts", "titel", for_table="ky035") to see available regions. niveau 1=5 regioner, niveau 2=11 landsdele, niveau 3=98 kommuner.
- visitation has TOT plus 5 categories. Filter visitation='TOT' for the total.
- Data ends at 2024-01-01. Only table with regional + visitation breakdown for helårsmodtagere.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
