table: fact.ky031
description: Kontanthjælp (personer og beløb) efter enhed, visitation, alder, køn og tid
measure: indhold (unit -)
columns:
- tal: values [1=Berørte personer (antal), 2=Helårsmodtagere (antal), 3=Udbetalt beløb (1.000 kr.)]
- visitation: values [TOT=I alt, 1=Jobparat, 2=Aktivitetsparat, 3=Åbenlyst uddannelsesparat, 4=Uddannelsesparat, 9=Uoplyst]
- alder: values [TOT=Alder i alt, 16-24=16-24 år, 25-29=25-29 år, 30-39=30-39 år, 40-49=40-49 år, 50-59=50-59 år, 6099=60 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- tid: date range 2007-01-01 to 2024-01-01
notes:
- tal is a measurement selector with 3 values: 1=Berørte personer (antal), 2=Helårsmodtagere (antal), 3=Udbetalt beløb (1.000 kr.). Always filter to one value — failure to do so triples all counts.
- Helårsmodtagere (tal=2) is an annual full-time-equivalent measure (person-years), not a headcount. Berørte personer (tal=1) counts unique individuals touched during the period.
- Data ends at 2024-01-01. For current data use ky13 (no age/gender) or ky01 (persons only).
- All category columns have TOT rows (visitation=TOT, alder=TOT, kon=TOT). Filter all to avoid overcounting.
