table: fact.pen123
description: Ydelse til førtidspensionister med bopæl i Danmark efter køn, alder, ydelsestype, familietype, enhed og tid
measure: indhold (unit -)
columns:
- kon: values [MOK=Mænd og kvinder i alt, M=Mænd, K=Kvinder]
- alder: values [1869=Alle 18-69 årige, 1824=18-24 år, 2529=25-29 år, 3034=30-34 år, 3539=35-39 år, 4044=40-44 år, 4549=45-49 år, 5054=50-54 år, 5559=55-59 år, 6064=60-64 år, 6569=65-69 år]
- ydelsestype: values [40=Førtidspension i alt, 41=Højeste førtidspension (brøkpension undtaget) ordning før 2003, 42=Mellemste førtidspension (brøkpension undtaget) ordning før 2003, 43=Almindelig/forhøjet almindelig førtidspension (brøkpension undtaget) ordning før 2003, 44=Førtidspension (brøkpension undtaget) ordning efter 2003, 45=Brøkpension til førtidspensionister, 55=Seniorpension, 60=Tidlig pension]
- famtype: values [FAMIALT=Familier i alt, ENL=Enlige, PAF=Parfamilier]
- enhed: values [7=Gennemsnitlig ydelse (kr.), 71=Alle modtagere (antal personer)]
- tid: date range 2021-01-01 to 2025-06-01

notes:
- enhed is a measurement selector — every dimension combination exists twice. Always filter to one enhed: 7=Gennemsnitlig ydelse (kr./måned), 71=Alle modtagere (antal). Never sum across both.
- famtype is an independent dimension: FAMIALT=total, ENL=enlige, PAF=parfamilier. Filter famtype='FAMIALT' for aggregate queries.
- alder=1869 is the total (18–69 årige); individual 5-year bands run from 1824 to 6569. ydelsestype=40 is the total — do not sum 40 with any of 41–45 or 55/60.
- ydelsestype covers ordning før 2003 (41, 42, 43) and ordning efter 2003 (44), plus brøkpension (45), seniorpension (55), and tidlig pension (60). For a modern snapshot use ydelsestype=40 (total) or ydelsestype=44 (post-2003 scheme).
- No regional breakdown in this table. For national average førtidspension amounts: WHERE kon='MOK' AND alder='1869' AND ydelsestype='40' AND famtype='FAMIALT' AND enhed=7.
