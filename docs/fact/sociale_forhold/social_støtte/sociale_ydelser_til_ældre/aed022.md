table: fact.aed022
description: Hjemmehjælp, visiteret tid efter område, ydelsestype, alder, køn og tid
measure: indhold (unit Timer pr. uge)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 3]
- ydelsestype: values [450=Visiterede timer i alt, 500=Visiterede timer til personlig pleje, i alt, 550=Visiterede timer til praktisk hjælp, i alt, 600=Visiterede timer til modtagere der udelukkende modtager personlig pleje, 650=Visiterede timer til modtagere der udelukkende modtager praktisk hjælp, 700=Visiterede timer til modtagere af både personlig pleje og praktisk hjælp i alt, 750=Visiterede timer til personlig pleje til modtagere af både personlig pleje og praktisk hjælp, 800=Visiterede timer til praktisk hjælp til modtagere af både personlig pleje og praktisk hjælp, 900=Visiterede timer til modtagere, der udelukkende modtager madservice (2023-)]
- alder: values [50=Alder i alt, 100=Under 65 år, 200=65-66 år, 300=67-69 år, 400=70-74 år, 500=75-79 år, 600=80-84 år, 700=85-89 år, 800=90 år og derover]
- koen: values [100=Mænd og kvinder i alt, 200=Mænd, 300=Kvinder]
- tid: date range 2008-01-01 to 2024-01-01
dim docs: /dim/nuts.md

notes:
- omrade joins dim.nuts with both niveau 1 (5 regioner) and niveau 3 (98 kommuner) present. Filter WHERE d.niveau=3 for kommuner, d.niveau=1 for regioner. omrade=0 is national total (not in dim).
- ydelsestype has a nested hierarchy — never sum across values: 450=i alt includes everything; 500=personlig pleje i alt = 600+750; 550=praktisk hjælp i alt = 650+800; 700=both types i alt includes 750+800. For total visited hours, filter ydelsestype=450. For a specific care type breakdown, pick one level.
- ydelsestype=900 (madservice) only available from 2023.
- alder=50 is total. Individual age bands (100-800) are exhaustive and non-overlapping; filter alder=50 for the total.
- koen=100 is total. Filter koen=100 for gender-neutral totals.
- Compare with aed01 for leveret (delivered) vs visiteret (authorized) hours.
- Map: /geo/kommuner.parquet (niveau 3) or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
