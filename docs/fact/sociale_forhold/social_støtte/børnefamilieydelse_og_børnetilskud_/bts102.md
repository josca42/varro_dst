table: fact.bts102
description: Modtagere af børnefamilieydelser og børnetilskud efter tilskudsart, familietype, antal børn, enhed og tid
measure: indhold (unit -)
columns:
- tilskudsart: values [1000=Børneydelse, 1005=Ungeydelse, 1010=Ordinært børnetilskud (til enlige), 1015=Ekstra børnetilskud (ekstra til enlige), 1020=Særligt børnetilskud (døde forældre ellere ukendt far), 1025=Flerbørnstilskud (Flerlinger under 7 år), 1030=Mildlertidig børnetilskud (familier med nedsat kontanthjælpsydelse, 2019-2023)]
- famtype: values [AFA=Alle familier, EMB=Enlige med børn, PMB=Par med børn, UBF=Uden børn på folkeregisteradresse mv.]
- antborn: values [0=Familier i alt, 1=1 barn, 2=2 børn, 3=3 børn, 41=4 eller flere børn]
- tal: values [30=Familier der modtager ydelsen (antal), 35=Gennemsnitlig udbetaling for familier med tilskudsarten (kr.), 40=Andel af familier, der modtager ydelsen (promille)]
- tid: date range 2017-01-01 to 2024-10-01

notes:
- tal is a measurement selector: 30 = familier der modtager ydelsen (antal), 35 = gennemsnitlig udbetaling (kr.), 40 = andel familier der modtager ydelsen (promille). Always filter to one tal value — they are three independent measures, never to be summed across.
- famtype='AFA' and antborn='0' are the totals. UBF (uden børn på folkeregisteradresse) appears because some benefits apply even when children aren't registered at the recipient's address.
- antborn here counts by number of children per family (unlike bts101 where it counts per person). antborn=0 = familier i alt.
- tilskudsart 1030 (midlertidig) only existed 2019–2023 — no data outside that range.
- This table has no regional breakdown (use bts101 for omrade splits). bts102 adds famtype (familiestruktur) instead.
