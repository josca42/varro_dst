table: fact.bu28
description: Samlede offentlige netto-driftsudgifter til udsatte børn og unge efter foranstaltning og tid
measure: indhold (unit Mio. kr.)
columns:
- foran: values [0=I alt, 410=Særlige dagtilbud og særlige klubber, 420=Plejefamilier og opholdssteder (-2015), 421=Opholdssteder for børn og unge (2016-), 422=Plejefamilier (2016-), 430=Forbyggende foranstaltninger, 440=Døgninstitutioner, 450=Sikrede døgninstitutioner, 460=Afgørelser om bekæmpelse af ungdomskriminalitet (2020-)]
- tid: date range 2002-01-01 to 2024-01-01

notes:
- Simple two-column table (foran + tid). No geographic breakdown.
- foran code 0 = I alt (total). Codes 410-460 are expenditure categories. Note that 420 (plejefamilier og opholdssteder) was split into 421 and 422 from 2016 — creating a break in the series for those categories.
- This is the only table covering expenditure; goes back to 2002. For long-run cost trends, no regional detail is available.
