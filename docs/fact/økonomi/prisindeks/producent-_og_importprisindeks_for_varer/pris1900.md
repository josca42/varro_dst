table: fact.pris1900
description: Prisindeks for indenlandsk vareforsyning; årsgennemsnit efter indekstype og tid
measure: indhold (unit Indeks)
columns:
- indekstype: values [1=Prisindeks for indenlandsk vareforsyning i alt]
- tid: date range 1876-01-01 to 2024-01-01

notes:
- This is a simple aggregate annual series — only 149 rows, one indekstype value (total), annual averages from 1876 to 2024. No filters needed beyond choosing a year range.
- Use this table only for long historical perspective on the overall domestic supply price level. For monthly data, industry breakdowns, or market splits, use the other tables in this subject.
- Example: SELECT tid, indhold FROM fact.pris1900 ORDER BY tid.
