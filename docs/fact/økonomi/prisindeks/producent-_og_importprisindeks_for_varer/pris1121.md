table: fact.pris1121
description: Prisindeks for indenlandsk vareforsyning efter varegruppe, enhed og tid
measure: indhold (unit Indeks)
columns:
- varegr: values [1=Prisindeks for indenlandsk vareforsyning i alt, 200=02 Kød og spiseligt slagteaffald, 300=03 Fisk og krebsdyr, 400=04 Mælk og mejeriprodukter mv., 500=05 Diverse produkter af animalsk oprindelse ... 9000=90 Optiske instr. mv., måle-, kontrolinstr. mv., 9100=91 Ure og dele dertil, 9400=94 Møbler samt visse belysningsartikler, 9500=95 Legetøj, spil, sportsartikler og dele dertil, 9600=96 Andre varer (2019- )]
- tal: values [100=Indeks, 200=Ændring i forhold til måneden før (pct.), 300=Ændring i forhold til samme måned året før (pct.)]
- tid: date range 1981-01-01 to 2025-09-01

notes:
- tal is a measurement selector that triples the data — always filter to one value: tal='100' (Indeks level), tal='200' (month-on-month change %), tal='300' (year-on-year change %).
- varegr='1' is the aggregate total (Prisindeks for indenlandsk vareforsyning i alt). All other varegr values are commodity group breakdowns. For a simple overall index, filter varegr=1.
- varegr uses a custom SITC-inspired product classification with 93 groups — not dim.db. No dim join needed or available. Use ColumnValues("pris1121", "varegr") to browse all groups with labels.
- This is the oldest and longest series in this subject (from 1981), classified by commodity type (CN/SITC). Use this table when you need long historical series by product group. For branch/industry breakdown instead, use pris4621 (from 2014) or pris4321 (from 2005).
