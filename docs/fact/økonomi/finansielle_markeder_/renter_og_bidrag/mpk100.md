table: fact.mpk100
description: Effektiv rente af statsobligationer efter land og tid
measure: indhold (unit Pct. pro anno)
columns:
- land: values [400=Belgien, 410=Bulgarien, 420=Danmark, 430=Estland, 440=Finland ... 820=Sydkorea, 830=Tyrkiet, 840=Australien, 850=New Zealand, 860=Euro-dollar-renten]
- tid: date range 1989-01-01 to 2024-01-01
notes:
- land uses 3-digit numeric country codes (e.g. 420=Danmark, 400=Belgien), not ISO alpha codes. Use ColumnValues("mpk100", "land") to browse the full country list.
- Annual data from 1989. Each land/year combination is a single government bond effective yield — no other dimensions to filter.
- Not all countries have data for all years. Missing rows mean no data, not zero.
