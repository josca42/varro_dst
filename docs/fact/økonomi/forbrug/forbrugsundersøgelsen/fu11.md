notes:
- bagopl is a measurement-type selector — each value is a different statistic (213=antal husstande i undersøgelsen, 214=husstande i Danmark (1.000), 255=personer i Danmark (1.000), 256=personer pr. husstand, 251=voksne, 252=børn, 250=andel boligejere (pct), 253=andel m. kvindelig hovedperson (pct), 254=alder på hovedperson, 261=forbrugsenheder, 307=areal m2). Always filter to exactly one bagopl — mixing them combines incompatible units (antal, procent, m2).
- husgrp covers 37 groups spanning multiple classification schemes: husstandstype (2001, 2010-2070), socioøkonomisk (3010-3080), indkomst (4015-4055), boligform (5010-5060), region (6010-6050), alder (7010-7050). Code 2001 = Gennemsnitshusstand (overall). These groupings are not mutually exclusive across schemes — pick one scheme at a time.
- No dim joins. This table is about survey demographics and household composition, not consumption amounts.
