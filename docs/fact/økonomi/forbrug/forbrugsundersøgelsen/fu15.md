table: fact.fu15
description: Forbrug efter forbrugsgruppe, samlet indkomst, prisenhed og tid
measure: indhold (unit Kr. pr. husstand)
columns:
- konsumgrp: join dim.ecoicop on konsumgrp=kode::text [approx]; levels [1, 2]
- samind: values [2001=Gennemsnitshusstand, 4015=Under 250.000 kr., 4025=250.000 - 449.999 kr., 4035=450.000 - 699.999 kr., 4045=700.000 - 999.999 kr., 4055=1.000.000 kr. og derover]
- prisenhed: values [AARPRIS=Løbende priser, 08PRIS=Faste priser]
- tid: date range 2015-01-01 to 2023-01-01
dim docs: /dim/ecoicop.md

notes:
- prisenhed MUST be filtered (AARPRIS or 08PRIS) — doubles all rows.
- samind must be filtered to one value: 2001=Gennemsnitshusstand, 4015=Under 250.000 kr., 4025=250-450k, 4035=450-700k, 4045=700k-1M, 4055=1M+. These are mutually exclusive income brackets (not hierarchical). 2001 is the overall average, not a sum of the income groups.
- konsumgrp: same ECOICOP dotted decimal scheme as fu13 (niveau 1 and 2, 62 codes). Same join formula and same unmatched codes.
