table: fact.fu14
description: Forbrug efter forbrugsgruppe, socioøkonomisk status, prisenhed og tid
measure: indhold (unit Kr. pr. husstand)
columns:
- konsumgrp: join dim.ecoicop on konsumgrp=kode::text [approx]; levels [1, 2]
- socio: values [2001=Gennemsnitshusstand, 3010=Selvstændig, 3020=Lønmodtager på højeste niveau, 3030=Lønmodtager på mellemniveau, 3040=Lønmodtager på grundniveau, 3050=Arbejdsløs, 3060=Uddannelsessøgende, 3070=Pensionist,efterlønmodtager, 3080=Ude af erhverv i øvrigt]
- prisenhed: values [AARPRIS=Løbende priser, 08PRIS=Faste priser]
- tid: date range 2015-01-01 to 2023-01-01
dim docs: /dim/ecoicop.md

notes:
- prisenhed MUST be filtered (AARPRIS or 08PRIS) — doubles all rows.
- socio must be filtered to one value: 2001=Gennemsnitshusstand, 3010=Selvstændig, 3020=Lønmodtager højeste niveau, 3030=Lønmodtager mellemniveau, 3040=Lønmodtager grundniveau, 3050=Arbejdsløs, 3060=Uddannelsessøgende, 3070=Pensionist/efterlønmodtager, 3080=Ude af erhverv i øvrigt.
- konsumgrp: same ECOICOP dotted decimal scheme as fu13 (niveau 1 and 2, 62 codes). Same join formula as fu13. Same unmatched codes.
