table: fact.ej67
description: Prisindeks for ejendomssalg efter område, ejendomskategori, enhed og tid
measure: indhold (unit Indeks)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2]
- ejendomskate: join dim.ejendom on ejendomskate=kode; levels [1, 2]
- tal: values [100=Indeks, 315=Ændring i forhold til året før (pct.)]
- tid: date range 1992-01-01 to 2024-01-01
dim docs: /dim/ejendom.md, /dim/nuts.md

notes:
- `tal` is a measurement selector: 100=Indeks, 315=YoY%. Always filter to one value.
- `omrade` joins dim.nuts at niveau 1 (5 regioner) and niveau 2 (11 landsdele). Code 0 = "Hele landet" (not in dim). Filter `d.niveau` to pick one level.
- `ejendomskate` only has 3 property types in this table. Use ColumnValues("ejendom", "titel", for_table="ej67") to see which.
- Annual price index with regional breakdown from 1992. Ends 2024. For quarterly with regions, use ej56 (current to 2025).
- Map: /geo/regioner.parquet (niveau 1) or /geo/landsdele.parquet (niveau 2) — merge on omrade=dim_kode. Exclude omrade=0.
