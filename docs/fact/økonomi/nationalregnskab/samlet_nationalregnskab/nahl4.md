table: fact.nahl4
description: 5 Danmark og udlandet (hovedposter) efter transaktion og tid
measure: indhold (unit Mio. kr.)
columns:
- transakt: values [P6D=P.6 Eksport af varer og tjenester, P61D=P.61 Eksport af varer, P62D=P.62 Eksport af tjenester, D1K=D.1 Modt. Aflønning af ansatte, D3K=D.3 Modt. Subsidier ... B111D=Varebalance, B112D=Tjenestebalance, B11D=B.11 Udlandskontoens vare- og tjenestebalance, B12D=B.12 Saldo på udlandskontoens løbende poster, B9D=B.9 Fordringserhvervelse, netto]
- tid: date range 1966-01-01 to 2024-01-01
notes:
- No prisenhed column — all values are in løbende priser only. Single measure per transakt×tid.
- transakt codes cover Denmark's current account with the rest of the world. Balance codes (B11D, B12D, B9D) are net saldos — do not sum them with underlying flows.
- 'Hoved' (main posts) annual table from 1966. Quarterly version with saeson: nkhl4. More detail from 1995: naho4 (oversigt) and nahd4 (detaljeret).
