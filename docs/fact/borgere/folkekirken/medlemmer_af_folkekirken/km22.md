table: fact.km22
description: Ind- og udmeldelser af folkekirken efter provsti, alder, folkekirkemedlemsskab og tid
measure: indhold (unit Antal)
columns:
- provsti: values [101=1-01 Københavns Vor Frue Provsti, 102=1-02 Københavns Holmens Provsti, 103=1-03 Østerbro Provsti, 104=1-04 Nørrebro Provsti, 105=1-05 Vesterbro Provsti ... 1004=10-04 Fredericia Provsti, 1005=10-05 Kolding Provsti, 1006=10-06 Vejle Provsti, 1007=10-07 Hedensted Provsti, 9999=99-99 Uoplyst provsti]
- alder: values [0=0 år, 1=1 år, 2=2 år, 3=3 år, 4=4 år ... 121=121 år, 122=122 år, 123=123 år, 124=124 år, 125=125 år]
- fkmed: values [F=Indmeldt i Folkekirken, U=Udmeldt af Folkekirken]
- tid: date range 2007-01-01 to 2025-04-01

notes:
- Provsti-level equivalent of km2 (movements, not stock). Same fkmed semantics: F=Indmeldt (joined), U=Udmeldt (left). NOT membership status.
- Quarterly data — sum across quarters for annual totals.
- alder uses individual years 0–125 (unlike km2 which uses 5-year bands). No IALT total. Sum all alder rows for aggregate by provsti.
- Net change: SUM(CASE WHEN fkmed='F' THEN indhold ELSE -indhold END) grouped by provsti/period.
