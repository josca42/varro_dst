table: fact.km11
description: Befolkningen den 1. i kvartalet efter provsti, folkekirkemedlemsskab og tid
measure: indhold (unit Antal)
columns:
- provsti: values [101=1-01 Københavns Vor Frue Provsti, 102=1-02 Københavns Holmens Provsti, 103=1-03 Østerbro Provsti, 104=1-04 Nørrebro Provsti, 105=1-05 Vesterbro Provsti ... 1004=10-04 Fredericia Provsti, 1005=10-05 Kolding Provsti, 1006=10-06 Vejle Provsti, 1007=10-07 Hedensted Provsti, 9999=99-99 Uoplyst provsti]
- fkmed: values [F=Medlem af Folkekirken, U=Ikke medlem af Folkekirken]
- tid: date range 2007-01-01 to 2025-07-01

notes:
- Provsti-level equivalent of km1 (quarterly). 111 distinct provsties including 9999='Uoplyst provsti'.
- No dim join for provsti — codes are inline. Use ColumnValues("km11", "provsti") for the full list.
- Quarterly data: filter WHERE EXTRACT(MONTH FROM tid)=1 for annual January snapshots.
- fkmed: F=Medlem, U=Ikke-Medlem. For membership rate: SUM(CASE WHEN fkmed='F' THEN indhold END)::float / SUM(indhold).
