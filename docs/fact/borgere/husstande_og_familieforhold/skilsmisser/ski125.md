notes:
- bopaegt1 and bopaegt2 join dim.nuts at niveau=3 (99 kommuner). Use ColumnValues("nuts", "titel", for_table="ski125") to see available codes.
- Code 0 = "Hele landet" (national aggregate) and code 997 = "Uoplyst bopæl" — neither is in dim.nuts. Use bopaegt1='0' AND bopaegt2='0' to get the national total without a JOIN; filter WHERE bopaegt1 NOT IN ('0','997') to work with kommune-level data only.
- alderaegt1 and alderaegt2 are individual ages (9915=Under 15, 15–99, 100-=100+, 999=Uoplyst). No aggregate total code — build age groups with CASE expressions in SQL.
- agtvar is marriage duration in years (0=under 1 year, 1–50, 99=Uoplyst). No aggregate total code.
- This table has 5 dimension columns. For a national total by age, filter bopaegt1='0' AND bopaegt2='0' to avoid inflating counts with all geographic permutations.
- Annual data (2007–2024). For national totals without geography, ski6 or ski125k are simpler.
