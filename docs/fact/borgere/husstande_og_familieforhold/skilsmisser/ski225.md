table: fact.ski225
description: Skilsmisser efter mandens eller ældste persons bopæl, Kvindens eller yngste persons bopæl, skilsmissemåned og tid
measure: indhold (unit Antal)
columns:
- bopaegt1: join dim.nuts on bopaegt1=kode [approx]; levels [3]
- bopaegt2: join dim.nuts on bopaegt2=kode [approx]; levels [3]
- skmdr: values [1=Januar, 2=Februar, 3=Marts, 4=April, 5=Maj, 6=Juni, 7=Juli, 8=August, 9=September, 10=Oktober, 11=November, 12=December, 999=Uoplyst måned]
- tid: date range 2007-01-01 to 2024-01-01
dim docs: /dim/nuts.md

notes:
- bopaegt1 and bopaegt2 join dim.nuts at niveau=3 (99 kommuner). Use ColumnValues("nuts", "titel", for_table="ski225") to see available codes.
- Code 0 = "Hele landet" (national aggregate), code 997 = "Uoplyst bopæl" — not in dim.nuts. Use bopaegt1='0' AND bopaegt2='0' for national totals; filter NOT IN ('0','997') for pure kommune data.
- skmdr is the month of divorce (1–12, 999=Uoplyst måned). Annual tid + skmdr together give month-level counts within each calendar year.
- Use this table for geography × month-of-year divorce patterns. For monthly totals without geography use ski6kvt.
- Map: /geo/kommuner.parquet — merge on bopaegt1=dim_kode (husband's kommune) or bopaegt2=dim_kode (wife's kommune). Exclude bopaegt1/bopaegt2 IN ('0','997').
