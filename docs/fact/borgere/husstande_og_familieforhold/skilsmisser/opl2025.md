table: fact.opl2025
description: Ægteskaber efter vielsesår, ægteskabets varighed, ægteskabsopløsning og tid
measure: indhold (unit -)
columns:
- vielsesaar: values [1970=1970, 1971=1971, 1972=1972, 1973=1973, 1974=1974 ... 2020=2020, 2021=2021, 2022=2022, 2023=2023, 2024=2024]
- agtvar: values [0=0 år, 1=1 år, 2=2 år, 3=3 år, 4=4 år ... 50=50 år, 51=51 år, 52=52 år, 53=53 år, 54=54 år]
- agtopl: values [1=Tilbageværende ægteskaber (antal), 2=Ægteskaber opløst ved skilsmisse (pr. 10.000 ægteskaber), 3=Ægteskaber opløst ved den ene parts død (pr. 10.000 ægteskaber), 4=Opløste ægteskaber i alt (pr. 10.000 ægteskaber)]
- tid: date range 2025-01-01 to 2025-01-01

notes:
- agtopl is a CRITICAL mixed-measure column — value 1 = count (tilbageværende ægteskaber), values 2/3/4 = rates per 10,000 marriages. These are incompatible; NEVER mix or sum across agtopl values. Always filter to a single value.
  - agtopl=1: remaining marriages (count) — use for cohort survival counts
  - agtopl=2: divorce rate (pr. 10,000 marriages)
  - agtopl=3: death-dissolution rate (pr. 10,000 marriages)
  - agtopl=4: total dissolution rate (pr. 10,000 marriages) — equals 2+3 conceptually
- vielsesaar = marriage year (1970–2024), agtvar = duration in years (0–54). Together they form a cohort survival matrix: track a vielsesaar cohort across increasing agtvar values.
- Single snapshot table — only tid=2025-01-01 exists. Do not filter or group by tid.
- Unique in this subject: the only table that lets you analyse marriage survival/dissolution rates by cohort and duration.
