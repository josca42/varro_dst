table: fact.bol201
description: Personer i boliger efter område, anvendelse, udlejningsforhold, ejerforhold, opførelsesår, alder, køn og tid
measure: indhold (unit Antal)
columns:
- amt: values [0=Hele landet, 84=Region Hovedstaden, 1=Landsdel Byen København, 101=København, 147=Frederiksberg ... 773=Morsø, 840=Rebild, 787=Thisted, 820=Vesthimmerlands, 851=Aalborg]
- anvendelse: values [125=Parcel/Stuehuse, 130=Række-, kæde- og dobbelthuse, 140=Etageboliger, 150=Kollegier, 160=Døgninstitutioner, 565=Fritidshuse, 570=Andet]
- udlforh: values [EJ=Beboet af ejer, LEJ=Beboet af lejer, UOPL=Uoplyst]
- ejer: values [10=Privatpersoner inkl I/S, 20=Almene boligselskaber, 30=A/S, ApS og andre selskaber, 41=Private andelsboligforeninger, SK=Offentlig myndighed, UOP2=Andet el uoplyst]
- opforelsesar: values [-1899=Før 1900, 1900-1919=1900-1919, 1920-1929=1920-1929, 1930-1939=1930-1939, 1940-1949=1940-1949 ... 2021=2021, 2022=2022, 2023=2023, 2024=2024, UOPL=Uoplyst]
- alder1: values [0-5=0-5 år, 6-11=6-11 år, 12-17=12-17 år, 18-24=18-24 år, 25-29=25-29 år, 30-39=30-39 år, 40-49=40-49 år, 50-59=50-59 år, 60-69=60-69 år, 70-79=70-79 år, 80-89=80-89 år, 90-=90 år og derover]
- koen: values [M=Mænd, K=Kvinder]
- tid: date range 2010-01-01 to 2025-01-01

notes:
- amt mixes 4 geographic levels in one column: 0=Hele landet, 81-85=5 Regioner, 1-11=11 Landsdele, 101-851=98 Kommuner. Always filter to ONE level — summing across all 115 values inflates counts 4x+. No dim join needed; filter inline (e.g. WHERE amt BETWEEN 81 AND 85 for regioner, WHERE amt >= 100 for kommuner).
- No total rows in any column. All values are categories, so SUM across multiple columns to aggregate. E.g. to get total persons by region: SUM(indhold) GROUP BY amt, filtered to the 5 regioner, with no other filters.
- ejer (property owner type) and opforelsesar (construction year) are unique to bol201 among the bol20x tables. opforelsesar has decade ranges for pre-2000 (e.g. '1900-1919') and individual years 2000-2024, plus UOPL.
- udlforh has only 3 values: EJ (owner-occupied), LEJ (rental), UOPL. No total — sum EJ+LEJ+UOPL to get all tenure types.
- Map: /geo/kommuner.parquet (amt 101–851), /geo/landsdele.parquet (amt 1–11), or /geo/regioner.parquet (amt 81–85) — filter amt to one level, then merge on amt=dim_kode. Exclude amt=0.
