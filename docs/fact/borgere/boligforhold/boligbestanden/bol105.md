table: fact.bol105
description: Boliger efter område, beboertype, anvendelse, opvarmning, toiletforhold, badeforhold, husstandstype, antal børn og tid
measure: indhold (unit Antal)
columns:
- amt: join dim.nuts on amt=kode; levels [1, 2, 3]
- bebo: values [1000=Boliger med CPR tilmeldte personer (beboede boliger), 2000=Boliger uden CPR tilmeldte personer (ubeboede boliger), 5000=Fritidshuse uden CPR tilmeldte personer (ubeboede fritidshuse)]
- anvendelse: values [125=Parcel/Stuehuse, 130=Række-, kæde- og dobbelthuse, 140=Etageboliger, 150=Kollegier, 160=Døgninstitutioner, 565=Fritidshuse, 570=Andet]
- opvarmning: values [FJ=Fjernvarme, CO=Centralvarme med olie, CN=Centralvarme m naturgas, CA=Centralvarme, ikke olie og naturgas, VP=Varmepumpe, ELV=Elvarme, OV=Øvrige ovne, UOPL=Uoplyst]
- toilet: values [1000618=WC i boligen, 1000616=WC udenfor boligen, 1000617=Andet/intet toilet, 1000619=Uoplyst Toiletforhold]
- bad: values [1000622=Badeværelser i boligen, 1000620=Adgang til badeværelse, 1000621=Ikke bad eller adgang til bad, 1000623=Uoplyst Badeforhold]
- hustyp: values [1M=Enlig mand, 1K=Enlig kvinde, PAR=Ægtepar, PA=Et par i øvrigt, IHB=Ikke hjemmeboende børn under 18 år, AH=Andre husstande bestående af flere familier, FH=Flere husstande i boligen, UB=Uden beboelse]
- antborn: values [0=0 børn, 1=1 barn, 2=2 børn, 3=3 børn, 4=4 børn, 5-=5 børn og derover]
- tid: date range 2010-01-01 to 2025-01-01
dim docs: /dim/nuts.md

notes:
- amt joins dim.nuts. kode=0 (Hele landet) not in dim — use WHERE amt=0 for national totals. niveau 1=5 regioner, 2=11 landsdele, 3=98 kommuner.
- No aggregate codes for any categorical column. SUM explicitly to aggregate across dimensions.
- toilet/bad values use long numeric codes (e.g. 1000618). Use exact codes as listed.
- This table combines facility info (opvarmning, toilet, bad) with household composition (hustyp, antborn) in one table. Use it when you need to cross-tabulate housing conditions against household type — otherwise bol102 (with more facility columns) or bol104 (with udlforh) may be more appropriate.
- hustyp='IHB' (ikke hjemmeboende børn under 18 år) is a special category — see bol104 notes.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on amt=dim_kode. Exclude amt=0.
