table: fact.bol101
description: Boliger efter område, beboertype, anvendelse, udlejningsforhold, ejerforhold, opførelsesår og tid
measure: indhold (unit Antal)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- bebo: values [1000=Boliger med CPR tilmeldte personer (beboede boliger), 2000=Boliger uden CPR tilmeldte personer (ubeboede boliger), 5000=Fritidshuse uden CPR tilmeldte personer (ubeboede fritidshuse)]
- anvendelse: values [125=Parcel/Stuehuse, 130=Række-, kæde- og dobbelthuse, 140=Etageboliger, 150=Kollegier, 160=Døgninstitutioner, 565=Fritidshuse, 570=Andet]
- udlforh: values [EJ=Beboet af ejer, LEJ=Beboet af lejer, IB=Ikke benyttet bolig, UOPL=Uoplyst]
- ejer: values [10=Privatpersoner inkl I/S, 20=Almene boligselskaber, 30=A/S, ApS og andre selskaber, 41=Private andelsboligforeninger, SK=Offentlig myndighed, UOP2=Andet el uoplyst]
- opforelsesar: values [-1899=Før 1900, 1900-1919=1900-1919, 1920-1929=1920-1929, 1930-1939=1930-1939, 1940-1949=1940-1949 ... 2021=2021, 2022=2022, 2023=2023, 2024=2024, UOPL=Uoplyst]
- tid: date range 2010-01-01 to 2025-01-01
dim docs: /dim/nuts.md

notes:
- omrade joins dim.nuts. kode=0 (Hele landet) is the national aggregate — not in the dim table, use WHERE omrade='0' directly for national totals. ColumnValues("nuts", "titel", for_table="bol101") shows the 114 joinable regions. niveau 1=5 regioner, 2=11 landsdele, 3=98 kommuner.
- None of the 6 categorical dimensions have a total/aggregate code. To get totals across all building types (anvendelse), tenant types (bebo), ownership categories (udlforh, ejer), or construction periods (opforelsesar), SUM all values in that dimension explicitly.
- opforelsesar uses period ranges for pre-2005 buildings (e.g. '1950-1959', '1900-1919', '-1899') and individual years from 2005 onwards ('2005', '2006', ..., '2024'). No IALT code. To get all periods, omit the filter or SUM all values.
- Typical query pattern: JOIN dim.nuts for geography, then filter or group by bebo, anvendelse, udlforh, ejer, opforelsesar as needed. Filtering all unwanted dimensions to a single value avoids overcounting.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
