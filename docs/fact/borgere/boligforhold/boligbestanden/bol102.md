table: fact.bol102
description: Boliger efter område, beboertype, anvendelse, opførelsesår, opvarmning, toiletforhold, badeforhold, køkkenforhold og tid
measure: indhold (unit Antal)
columns:
- amt: values [0=Hele landet, 84=Region Hovedstaden, 1=Landsdel Byen København, 101=København, 147=Frederiksberg ... 773=Morsø, 840=Rebild, 787=Thisted, 820=Vesthimmerlands, 851=Aalborg]
- bebo: values [1000=Boliger med CPR tilmeldte personer (beboede boliger), 2000=Boliger uden CPR tilmeldte personer (ubeboede boliger), 5000=Fritidshuse uden CPR tilmeldte personer (ubeboede fritidshuse)]
- anvendelse: values [125=Parcel/Stuehuse, 130=Række-, kæde- og dobbelthuse, 140=Etageboliger, 150=Kollegier, 160=Døgninstitutioner, 565=Fritidshuse, 570=Andet]
- opforelsesar: values [-1899=Før 1900, 1900-1919=1900-1919, 1920-1929=1920-1929, 1930-1939=1930-1939, 1940-1949=1940-1949 ... 2021=2021, 2022=2022, 2023=2023, 2024=2024, UOPL=Uoplyst]
- opvarmning: values [FJ=Fjernvarme, CO=Centralvarme med olie, CN=Centralvarme m naturgas, CA=Centralvarme, ikke olie og naturgas, VP=Varmepumpe, ELV=Elvarme, OV=Øvrige ovne, UOPL=Uoplyst]
- toilet: values [1000618=WC i boligen, 1000616=WC udenfor boligen, 1000617=Andet/intet toilet, 1000619=Uoplyst Toiletforhold]
- bad: values [1000622=Badeværelser i boligen, 1000620=Adgang til badeværelse, 1000621=Ikke bad eller adgang til bad, 1000623=Uoplyst Badeforhold]
- koekken: values [1000624=Eget køkken med afløb, 1000625=Adgang til fælles køkken, 1000626=Kogeinstallation i værelse, 1000627=Ingen fast kogeinstallation, 1000628=Uoplyst Køkkeninstallation]
- tid: date range 2010-01-01 to 2025-01-01

notes:
- amt joins dim.nuts despite being documented with inline values. Use JOIN dim.nuts ON f.amt = d.kode. kode=0 is Hele landet (not in dim — use WHERE amt=0 directly for national totals). niveau 1=5 regioner, 2=11 landsdele, 3=98 kommuner (same as bol101/103/104/105).
- opforelsesar uses period ranges pre-2005 (e.g. '1950-1959') and individual years 2005+ ('2005', ..., '2024'). No aggregate code.
- No total codes for any categorical column (bebo, anvendelse, opforelsesar, opvarmning, toilet, bad, koekken). To get national totals across a dimension, SUM all its values.
- toilet/bad/koekken values use long numeric codes (e.g. 1000618=WC i boligen). Use the exact codes as shown in the column values — they are not meaningful numbers.
- Combines building facilities (opvarmning, toilet, bad, koekken) in one table — useful for housing quality analysis.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on amt=dim_kode. Exclude amt=0.
