table: fact.fod111
description: Gennemsnitsalder for fødende kvinder og nybagte fædre efter område, alder og tid
measure: indhold (unit Gns.)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 3]
- alder: values [610=Gennemsnitsalder for førstegangsfødende kvinder, 611=Gennemsnitsalder for samtlige fødende kvinder, 617=Gennemsnitsalder for førstegangs fædre, 612=Gennemsnitsalder for fædre til nyfødte]
- tid: date range 2006-01-01 to 2024-01-01
dim docs: /dim/nuts.md
notes:
- indhold is an average age — NEVER sum across alder values. Each alder code is an independent metric (same 4 as fod11). Always filter to one alder per query.
- omrade joins dim.nuts at niveau 1 (5 regioner) and niveau 3 (99 kommuner). Code '0' is the national total, not in dim.nuts. Filter d.niveau to pick one granularity.
- Sample query for average first-time mother age by region: JOIN dim.nuts ON omrade=kode WHERE d.niveau=1 AND alder='610'.
- Use ColumnValues("nuts", "titel", for_table="fod111") to see the 104 codes (5 regioner + 99 kommuner + code 0).
- Map: /geo/kommuner.parquet (niveau 3) or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
