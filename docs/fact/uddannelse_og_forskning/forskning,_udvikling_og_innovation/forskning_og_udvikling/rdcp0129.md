table: fact.rdcp0129
description: Ansatte inden for FoU efter sektor, køn, personaletype, årsværk og personale og tid
measure: indhold (unit Antal)
columns:
- sektor: values [10=I alt sektor, 100=Private erhverv, 110=Offentlige myndigheder/institutioner, 120=Højere læreanstalter og universitetshospitaler, 130=Private ikke profitdrevne organisationer]
- kon: values [TOT=I alt, K=Kvinder]
- perstyp: values [90=PERSONALE I ALT I FOU, 900=Forskere og andre specialister, 910=TEKNIKERE OG ØVRIGT PERSONALE I ALT, 920=Teknikere, 930=Øvrigt personale]
- aarvaerk: values [800=Ansatte omregnet til årsværk, 810=Ansatte]
- tid: date range 2008-01-01 to 2023-01-01
notes:
- aarvaerk is a selector column: 800=FTE (årsværk), 810=headcount (ansatte). Every sektor/kon/perstyp/tid combination appears twice. Always filter to one value — use 810 for person counts, 800 for FTE.
- kon has only TOT and K (no M code for men). To get men: either use kon='TOT' only, or derive men = TOT − Kvinder in SQL.
- perstyp hierarchy: 90=PERSONALE I ALT (total). 900=Forskere og andre specialister and 910=TEKNIKERE OG ØVRIGT PERSONALE I ALT sum to 90. Under 910: 920=Teknikere and 930=Øvrigt personale. Never sum 90 + its sub-codes.
- sektor='10' = I alt sektor (grand total). sektor 100–130 = sub-sectors.
- This is the cross-sector personnel table (both public and private). For public-sector detail with gender/field breakdown, use fouoff01/fouoff02. For business-sector detail, use forsk113/forsk114.
