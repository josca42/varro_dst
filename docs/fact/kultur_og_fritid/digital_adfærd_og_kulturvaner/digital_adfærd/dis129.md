table: fact.dis129
description: Aktive internetabonnementer efter antal og andel, privat og erhverv og tid
measure: indhold (unit Antal)
columns:
- antalandel: values [163750=Antal, 163755=Markedsandel]
- privat: values [163710=I alt, 163700=Erhvervsabonnementer, 163702=Erhvervsabonnementer brugt af erhverv, 163703=Erhvervsabonnementer brugt af private, 163705=Privatabonnementer]
- tid: date range 2002-07-01 to 2024-07-01
notes:
- antalandel is a measurement selector: '163750'=Antal (count), '163755'=Markedsandel (market share %). These are different measures — never mix or sum across antalandel values. Always filter to one.
- privat='163710' is I alt. The other privat values are: '163700'=all business subscriptions, '163702'=business subs used by businesses, '163703'=business subs used by private individuals, '163705'=consumer subscriptions. Note that 163700 = 163702 + 163703, so filtering to 163710 already avoids double-counting.
- Long time series: July 2002 to July 2024, with mid-year observations.
