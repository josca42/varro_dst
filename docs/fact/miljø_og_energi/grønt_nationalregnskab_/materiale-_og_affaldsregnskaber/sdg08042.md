table: fact.sdg08042
description: Indikator: Indenlandsk materialeanvendelse efter område, måleenhed og tid
measure: indhold (unit Tons pr. indb.)
columns:
- omr20: values [00X=I alt]
- maleenhed: values [DMC=Indenlandsk materialeanvendelse (mio. tons), DMCCAP=Indenlandsk materialeanvendelse (tons pr. indbygger), DMCGDP=Indenlandsk materialeanvendelse (tons pr. mia. kr. BNP)]
- tid: date range 2010-01-01 to 2023-01-01
notes:
- maleenhed is a measurement-unit selector (3 values): DMC=Indenlandsk materialeanvendelse i mio. tons, DMCCAP=tons pr. indbygger, DMCGDP=tons pr. mia. kr. BNP. Always filter to one — the three units are incommensurable.
- omr20 only ever contains '00X' (I alt) — this table has no regional breakdown. Denmark as a whole only.
- Useful as a headline SDG indicator for domestic material consumption per capita or per GDP. For material type breakdown use mrm2 instead.
