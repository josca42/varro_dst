table: fact.tema9010
description: Udledning fra landbrug, skovbrug og fiskeri efter emissionstype og tid
measure: indhold (unit Mio ton CO2 -ækvivalenter)
columns:
- emtype8: values [450=Kuldioxid (CO2), ekskl. fra afbrænding af biomasse, 452=Lattergas (N2O), 454=Metan (CH4)]
- tid: date range 1990-01-01 to 2024-01-01

notes:
- 3 GHG types for the agriculture/forestry/fishing sector, all in Mio ton CO2-equivalents. The 3 emtype8 values are additive.
- Agriculture is the dominant source of N2O and CH4 in Denmark. This table drills into that. For other sectors' GHGs, use drivhus (which covers all sectors but with different emtype8 codes).
