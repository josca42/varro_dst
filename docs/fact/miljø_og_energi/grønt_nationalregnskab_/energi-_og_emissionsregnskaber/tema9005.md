description: Udledning af drivhusgas efter branche og tid
measure: indhold (unit 1.000 ton)
columns:
- erhverv: values [421=Landbrug, skovbrug og fiskeri, 422=Industri, 423=Forsyningsvirksomhed, 424=Øvrige virksomheder]

notes:
