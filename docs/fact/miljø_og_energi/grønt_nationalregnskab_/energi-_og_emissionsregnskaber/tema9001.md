table: fact.tema9001
description: Økonomisk vækst og udledning af drivhusgas efter enhed og tid
measure: indhold (unit Indeks)
columns:
- enhed: values [0=Udledning fra dansk økonomi, inkl. danske transportvirksomheder i udlandet , 4=Udledning i Danmark , 6=Økonomisk vækst (BNP)]
- tid: date range 1990-01-01 to 2024-01-01

notes:
- enhed is a series selector — each tid has 3 rows (GHG economy principle, GHG territorial, GDP growth). These are index series (1990=100), not absolute values. Do NOT sum across enhed values.
- For absolute GHG levels, use tema9007 (Mio ton) or drivhus (1.000 ton). Use tema9001 only for decoupling analysis (comparing GHG trend vs. GDP trend).
