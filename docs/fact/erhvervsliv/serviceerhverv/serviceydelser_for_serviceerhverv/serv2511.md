table: fact.serv2511
description: Solgte vikartimer efter branche og tid
measure: indhold (unit Timer)
columns:
- erhverv: values [TOT3=Timer i alt, T70020=It og telekommunikation, T70030=Andet kontorpersonale, T70040=Handelsbranchen, T70045=Produktion, T70055=Bygge/anlæg, T70060=Transport, lager og logistik, T70070=Hotel- og restaurationsbranchen, T70080=Sygeplejersker, T70090=Social- og sundhedsassistenter, T70100=Social- og sundhedshjælpere, T70110=Læger, T70120=Øvrigt personale inden for sygepleje og sundhed, T70130=Øvrige personalekategorier]
- tid: date range 2023-01-01 to 2023-01-01
notes:
- Single snapshot: only 2023 data. No time-series analysis possible.
- Measures hours (timer), not revenue — the companion revenue table is serv2502 (Vikarbureauer).
- erhverv total code is TOT3 (Timer i alt). Exclude it when summing: WHERE erhverv != 'TOT3'. TOT3 = exact sum of all other rows.
- The erhverv codes start with T (T70020, T70030...) matching the 70xxx service types in serv2502, but without recruitment of managers (70000) or job placement (70010) — those are placement services rather than hourly temp work.
- Use ColumnValues("serv2511", "erhverv") to see all 14 codes.
