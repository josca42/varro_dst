table: fact.serv2503
description: Reklamevirksomhed efter serviceydelser og tid
measure: indhold (unit 1.000 kr.)
columns:
- servyd: values [TOT=I alt, 60000=Salg af reklameplads i trykte medier på honorar- eller kontraktbasis, 60010=Salg af reklameplads eller -tid i radio og fjernsyn på honorar- eller kontraktbasis, 60020=Salg af reklameplads eller -tid på internettet på honorar- eller kontraktbasis, 60030=Salg af reklame i forbindelse med events, 60040=Andet salg af reklameplads eller -tid på honorar- eller kontraktbasis, 60060=Totalrådgivning vedr. reklame, 60070=Direct marketing og direct mailing, 60080=Reklamedesign og konceptudvikling, 60090=Anden reklamevirksomhed, 60100=Andre serviceydelser]
- tid: date range 2023-01-01 to 2023-01-01
notes:
- Single snapshot: only 2023 data. No time-series analysis possible.
- servyd contains 11 values including TOT (I alt), which is the aggregate. Exclude it when summing across service types: WHERE servyd != 'TOT'.
- For advertising revenue broken down by media channel (TV, internet, print, etc.) use serv2512 instead — serv2503 breaks down by type of advertising service, serv2512 by medium.
- Use ColumnValues("serv2503", "servyd") to see all 11 service type codes with labels.
