table: fact.serv2501
description: It-servicevirksomhed efter serviceydelser og tid
measure: indhold (unit 1.000 kr.)
columns:
- servyd: values [TOT=I alt, 20002=Udvikling af software applikationer (apps), 20003=Udvikling af computerspil til alle platforme, 20004=Udvikling af anden computer software, 20010=Konsulentbistand vedr. hardware ... 20160=Reparation af computere og ydre enheder, 20170=Videresalg af software, som virksomheden ikke selv har udviklet, 20180=Videresalg af hardware og andet udstyr, 20190=Videresalg af andre varer, 20200=Andre serviceydelser i.a.n.]
- tid: date range 2023-01-01 to 2023-01-01
notes:
- Single snapshot: only 2023 data. No time-series analysis possible.
- servyd contains 22 values including TOT (I alt), which is the aggregate of all service types. TOT ≈ sum of all other rows. Exclude it when summing across service types: WHERE servyd != 'TOT'. Forgetting this double-counts revenue.
- To get total IT-service revenue: SELECT indhold FROM fact.serv2501 WHERE servyd = 'TOT' (≈111 mio. kr. in 1.000 kr. units = ~111 bn. kr.).
- Use ColumnValues("serv2501", "servyd") to see all 22 service type codes with labels.
