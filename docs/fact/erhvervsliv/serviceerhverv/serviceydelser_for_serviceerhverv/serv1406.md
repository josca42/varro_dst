table: fact.serv1406
description: Bogføring- og revisionsvirksomhed efter serviceydelser, omsætning og eksport og tid
measure: indhold (unit 1.000 kr.)
columns:
- servyd: values [TOTSERV=I alt, 40000=Revisionsvirksomhed, 40010=Regnskabsanalyse, 40020=Udarbejdelse af regnskaber, 40030=Bogføring, 40040=Lønregnskaber, 40050=Andre regnskabstjenester, 40060=Skatterådgivning, inklusiv selvangivelse og selskabsskat, 40070=Tjenester ifm. konkurs og bestyrelse af konkursboer, 40080=Andre serviceydelser i.a.n.]
- omseks: values [TOTOMS=Omsætning, EKST=Eksport]
- tid: date range 2012-01-01 to 2022-01-01
notes:
- Multi-year time series: 2012–2022 (annual). The only tables in this subject with historical data.
- Two dimension columns require filtering: servyd and omseks. Always filter both to avoid overcounting.
- omseks: TOTOMS = total turnover, EKST = exports only. EKST is a subset of TOTOMS — they are NOT additive. For total revenue use omseks='TOTOMS'; for exports only use omseks='EKST'. Never sum TOTOMS + EKST.
- servyd total code is TOTSERV (I alt). Exclude it when aggregating: WHERE servyd != 'TOTSERV'. TOTSERV = exact sum of all other servyd rows.
- Sample query for total annual revenue: SELECT tid, indhold FROM fact.serv1406 WHERE servyd='TOTSERV' AND omseks='TOTOMS' ORDER BY tid.
- Use ColumnValues("serv1406", "servyd") to see all 10 accounting/audit service types.
