table: fact.serv1407
description: Anden virksomhedsrådgivning efter serviceydelser, omsætning og eksport og tid
measure: indhold (unit 1.000 kr.)
columns:
- servyd: values [TOTSERV=I alt, 50000=Rådgivning inden for public relations og kommunikation, 50010=Strategisk virksomhedsrådgivning, 50020=Rådgivning inden for det finansielle område (undtagen selskab), 50030=Rådgivning inden for markedsføring, 50040=Rådgivning inden for human resources, 50050=Rådgivning inden for produktion, 50060=Rådgivning inden for forsyningskæder og anden driftsledelse, 50070=Styring af virksomhedsprocesser, 50080=Anden projektstyring, 50090=Anden virksomhedsrådgivning, 50100=Varemærker og franchise, 50110=Uddannelse, 50120=Andre serviceydelser i.a.n.]
- omseks: values [TOTOMS=Omsætning, EKST=Eksport]
- tid: date range 2012-01-01 to 2022-01-01
notes:
- Multi-year time series: 2012–2022 (annual). The only tables in this subject with historical data (along with serv1406).
- Two dimension columns require filtering: servyd and omseks. Always filter both to avoid overcounting.
- omseks: TOTOMS = total turnover, EKST = exports only. EKST is a subset of TOTOMS — they are NOT additive. For total revenue use omseks='TOTOMS'; for exports only use omseks='EKST'. Never sum TOTOMS + EKST.
- servyd total code is TOTSERV (I alt). Exclude it when aggregating: WHERE servyd != 'TOTSERV'. TOTSERV = exact sum of all other servyd rows.
- Sample query for total annual revenue: SELECT tid, indhold FROM fact.serv1407 WHERE servyd='TOTSERV' AND omseks='TOTOMS' ORDER BY tid.
- Use ColumnValues("serv1407", "servyd") to see all 13 business consulting service categories.
