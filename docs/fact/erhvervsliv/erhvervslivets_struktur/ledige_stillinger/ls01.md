table: fact.ls01
description: Ledige stillinger efter branche, enhed, størrelse og tid
measure: indhold (unit -)
columns:
- branche: values [0=I alt, 2=2 Industri, råstofindvinding og forsyningsvirksomhed, 3=3 Bygge og anlæg, 4=4 Handel og transport mv., 5=5 Information og kommunikation, 6-7=6-7 Finansiering, forsikring og ejendomshandel, 8=8 Erhvervsservice]
- enhed: values [LS=Ledige stillinger (antal), ALS=Andel ledige stillinger (procent)]
- storrelse: values [0=Arbejdssteder i alt, 105=1-9 ansatte, 115=10-49 ansatte, 125=50-99 ansatte, 130=100 ansatte og derover, 135=Fiktive enheder]
- tid: date range 2010-01-01 to 2024-01-01

notes:
- Annual data (one row per year). For quarterly, use lsk01 instead.
- enhed is a measurement selector: LS=Ledige stillinger (antal), ALS=Andel ledige stillinger (procent). Always filter to one value — mixing them inflates counts. ALS is a percentage and must never be summed.
- branche code 0 = I alt (all industries combined). Codes 2-8 are individual industries. Summing all branche values double-counts (totals + components). Filter branche='0' for the national industry total or use individual codes 2-8 and exclude 0.
- storrelse code 0 = Arbejdssteder i alt (all sizes). Codes 105/115/125/130 are size bands. Code 135 = Fiktive enheder (virtual units, included in the storrelse=0 total). Filter storrelse=0 for the size-aggregated total.
- Simple national vacancy count: WHERE branche='0' AND storrelse=0 AND enhed='LS'.
