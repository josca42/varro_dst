table: fact.gf14
description: Generel firmastatistik efter region, enhed, firmastørrelse (fuldtidsansatte) og tid
measure: indhold (unit -)
columns:
- region: join dim.nuts on region=kode; levels [1]
- enhed: values [AFI=Firmaer (antal), AFU=Fuldtidsansatte (antal), OMS=Omsætning (mio kr.) ]
- firmstr: values [TOT=I alt, 0000=Ingen ansatte, 010=Under 10 ansatte, 101=10-49 ansatte, 102=50-249 ansatte, 103=250 ansatte og derover]
- tid: date range 2019-01-01 to 2023-01-01
dim docs: /dim/nuts.md

notes:
- region joins dim.nuts. Only niveau 1 codes are present (5 Danish regions). Code 0 = "Hele landet" (aggregate, not in dim.nuts). Straightforward join: `JOIN dim.nuts d ON f.region = d.kode WHERE d.niveau = 1`.
- enhed is a measurement selector (AFI=firmaer, AFU=fuldtidsansatte, OMS=omsætning). Always filter to one value.
- firmstr includes TOT plus 5 size buckets (same as gf13). Filter to TOT for overall totals.
- Map: /geo/regioner.parquet — merge on region=dim_kode. Exclude region=0.
