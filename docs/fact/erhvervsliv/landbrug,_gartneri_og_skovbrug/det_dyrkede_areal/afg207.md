table: fact.afg207
description: Bedrifter og arealer med udvalgte afgrøder efter område, areal med afgrøden, enhed og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode [approx]; levels [1, 2]
- arealafg: values [10000=Uden korn, 10005=0,1-4,9 ha med korn, 10010=5,0-9,9 ha med korn, 10015=10,0-14,9 ha med korn, 10020=15,0-19,9 ha med korn ... 10225=20,0-29,9 ha med græs og grøntfoder i omdriften, 10230=30,0-49,9 ha med græs og grøntfoder i omdriften, 10235=50,0-99,9 ha med græs og grøntfoder i omdriften, 10240=100,0 ha med græs og grøntfoder i omdriften og over, 10245=Alle bedrifter med græs og grøntfoder i omdriften]
- enhed: values [200=Ha med afgrøden, 400=Bedrifter ]
- tid: date range 2006-01-01 to 2024-01-01
dim docs: /dim/nuts.md
notes:
- omrade contains 12 values mixing two dim.nuts hierarchy levels (niveau 1: 5 regioner codes 81–85; niveau 2: 5 specific landsdele codes 4,7,8,9,10) plus two unmatched codes: 0=Hele landet and 15="Landsdelene Byen København, Københavns omegn og Nordsjælland" (a merged landsdel not in dim.nuts). Use ColumnValues("afg207","omrade") to get all labels directly. A dim.nuts join works for regioner/landsdele but drops rows 0 and 15.
- enhed is a measurement selector: 200=Ha med afgrøden, 400=Bedrifter. Every omrade × arealafg combination appears twice. Always filter to one enhed.
- arealafg encodes both crop type and farm-size class in one column. Each crop has its own range of ~10 codes (e.g. 10000–10045 for korn, 10050–10095 for hvede, 10100–10145 for raps, 10150–10195 for varig græs, 10200–10245 for græs og grøntfoder i omdrift). The highest code in each range (e.g. 10045="Alle bedrifter med korn") gives the crop total across all farm sizes. Use ColumnValues("afg207","arealafg") to browse all 50 codes.
- Example: total hectares with grain nationally in 2024: SELECT indhold FROM fact.afg207 WHERE tid='2024-01-01' AND omrade=0 AND enhed=200 AND arealafg=10045.
- Map: /geo/regioner.parquet (niveau 1, codes 81–85) or /geo/landsdele.parquet (niveau 2, codes 4,7,8,9,10) — merge on omrade=dim_kode. Exclude omrade IN (0, 15).
