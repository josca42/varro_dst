table: fact.bdf207
description: Bedrifter med forpagtning efter område, type, areal og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode [approx]; levels [1, 2]
- type: values [BFP=Bedrifter med forpagtning, ANTAL=Bedrifter (antal), AFP=Forpagtet areal (ha), AREAL=Samlet dyrket areal (ha)]
- areal1: values [AIALT=I alt, A1=Under 10,0 ha, A19=10,0 - 19,9 ha, A29=20,0 - 29,9 ha, A49=30,0 - 49,9 ha, A62=50,0 - 74,9 ha, A100=75,0 - 99,9 ha, A210=100,0 - 199,9 ha, A220=200,0 ha og derover]
- tid: date range 2006-01-01 to 2024-01-01
dim docs: /dim/nuts.md

notes:
- omrade joins dim.nuts with the same 12 codes as bdf11: niveau 1 (81-85), niveau 2 (4,7,8,9,10), plus aggregates 0 and 15. See bdf11 notes for join details.
- type has 4 distinct measurement types in one column: BFP=Bedrifter med forpagtning (count), ANTAL=Bedrifter total (count), AFP=Forpagtet areal (ha), AREAL=Samlet dyrket areal (ha). Every combination of other dimensions appears 4 times. Always filter to exactly ONE type value.
- areal1: AIALT = all farm sizes combined. Filter to AIALT unless you need a size breakdown.
- For total farms with leased land nationally: WHERE type='BFP' AND areal1='AIALT' AND omrade=0.
- Map: /geo/regioner.parquet (niveau 1, omrade 81-85) or /geo/landsdele.parquet (niveau 2, omrade 4,7,8,9,10 — 5 of 11 landsdele only) — merge on omrade=dim_kode. Exclude omrade IN (0, 15).
