table: fact.ani302
description: Mængde- og prisindeks for animalske landbrugsprodukter (kvartal) efter varegruppe, indekstype og tid
measure: indhold (unit -)
columns:
- varegr: values [ANATUR=Naturmælk, AOKO=Økologisk mælk, AKONSUM=Konsumæg, AKVAEG=Kvæg, ASVIN=Svin, AFJER=Fjerkræ, AFAAR=Får]
- indekstype: values [15=Mængdeindeks 2015=100, 25=Prisindeks 2015=100]
- tid: date range 2015-01-01 to 2025-04-01
notes:
- indekstype is a measurement selector: filter to ONE (mængde- or pris-indeks). Values are indices (2015=100), not summable across varegr.
- Quarterly frequency. More current than ani301 (up to 2025Q1 vs 2024 for annual). Fewer varegr (7) — no Heste, Pelsdyr, Vildt, Andre for quarterly.
- For annual aggregates use ani301; for monthly use ani303 (5 varegr only).
