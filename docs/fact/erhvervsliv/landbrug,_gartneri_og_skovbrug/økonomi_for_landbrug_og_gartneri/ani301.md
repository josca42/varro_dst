table: fact.ani301
description: Mængde- og prisindeks for animalske landbrugsprodukter (år) efter varegruppe, indekstype og tid
measure: indhold (unit -)
columns:
- varegr: values [ANATUR=Naturmælk, AOKO=Økologisk mælk, AKONSUM=Konsumæg, AKVAEG=Kvæg, ASVIN=Svin, AFJER=Fjerkræ, AHESTE=Heste, AFAAR=Får, APELS=Pelsdyr, AVILDT=Vildt, AANIAND=Andre animalske produkter]
- indekstype: values [15=Mængdeindeks 2015=100, 25=Prisindeks 2015=100]
- tid: date range 2015-01-01 to 2024-01-01
notes:
- indekstype is a measurement selector: 15=Mængdeindeks 2015=100, 25=Prisindeks 2015=100. Every (varegr, tid) combination appears twice. Always filter to ONE indekstype.
- Values are indices (2015=100), not quantities or prices. Do not sum across varegr — indices are not additive.
- Annual frequency. For quarterly data use ani302, for monthly use ani303. Note ani301 has more varegr (11) than ani302 (7) or ani303 (5) — Heste, Pelsdyr, Vildt, Andre animalske only available annually.
