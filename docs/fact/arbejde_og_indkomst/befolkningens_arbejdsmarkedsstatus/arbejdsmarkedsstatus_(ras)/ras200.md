table: fact.ras200
description: Erhvervs- og beskæftigelsesfrekvenser (ultimo november) efter område, herkomst, alder, køn, frekvens og tid
measure: indhold (unit Pct.)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- herkomst: values [0=I alt, 10=Personer med dansk oprindelse, 20=Indvandrere, 24=Indvandrere fra vestlige lande, 25=Indvandrere fra ikke-vestlige lande, 30=Efterkommere, 34=Efterkommere fra vestlige lande, 35=Efterkommere fra ikke-vestlige lande]
- alder: values [16-64=16-64 år, 16-65=16-65 år (2020 -), 1666=16-66 år (2022 -), 16-17=16-17 år, 18-19=18-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- beregning: values [BFK=Beskæftigelsesfrekvens, EFK=Erhvervsfrekvens]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/nuts.md

notes:
- beregning is a measurement selector (BFK=Beskæftigelsesfrekvens, EFK=Erhvervsfrekvens). Every combination of other dimensions is repeated once per beregning. Always filter to one: WHERE beregning = 'BFK'.
- omrade joins dim.nuts on omrade=kode. dim.nuts has 5 regioner (niveau 1), 11 landsdele (niveau 2), 99 kommuner (niveau 3). Filter WHERE d.niveau = N to pick granularity and avoid double-counting across levels.
- omrade='0' is the national total ("Hele landet") — not in dim.nuts. Use it directly without JOIN: WHERE f.omrade = '0'.
- alder uses range codes rather than totals: '16-64', '16-65', '1666' (note: 1666 = 16-66 år), etc. There is no TOT/IALT. The broad ranges overlap over time (age limit was extended): 16-64 pre-2020, 16-65 from 2020, 1666 from 2022.
- kon has TOT. For overall rates filter kon='TOT'.
- Sample query — BFK by region for 2022: SELECT f.omrade, d.titel, f.indhold FROM fact.ras200 f JOIN dim.nuts d ON f.omrade = d.kode WHERE d.niveau = 1 AND f.herkomst = '0' AND f.alder = '16-64' AND f.kon = 'TOT' AND f.beregning = 'BFK' AND f.tid = '2022-01-01';
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
