table: fact.ras201
description: Befolkningen (ultimo november) efter område, socioøkonomisk status, herkomst, alder, køn og tid
measure: indhold (unit Antal)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- socio: values [0=Beskæftigede, 50=Arbejdsløse, 55=Uden for arbejdsstyrken]
- herkomst: values [999=Personer med dansk oprindelse, 24=Indvandrere fra vestlige lande, 25=Indvandrere fra ikke-vestlige lande, 34=Efterkommere fra vestlige lande, 35=Efterkommere fra ikke-vestlige lande]
- alder: values [-15=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/nuts.md

notes:
- omrade joins dim.nuts on omrade=kode. dim.nuts has 5 regioner (niveau 1), 11 landsdele (niveau 2), 99 kommuner (niveau 3). Filter WHERE d.niveau = N to avoid double-counting across levels.
- omrade='0' is the national total — not in dim.nuts. Use without JOIN: WHERE f.omrade = '0'.
- kon has NO total row — only M and K. To get both genders combined, SUM across M and K.
- herkomst uses different codes than ras200: 999=Dansk oprindelse (not code 10), and only westlig/ikke-vestlig sub-groups (no aggregate indvandrer/efterkommer totals). There is no "I alt" code.
- socio has only 3 values (0=Beskæftigede, 50=Arbejdsløse, 55=Uden for arbejdsstyrken) — simpler than ras202/ras209. Sum across all three to get total population.
- alder has all age groups including the catch-all ranges (-15=Under 16, 67-=67+) but no grand total. Sum age groups explicitly.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
