table: fact.ras205
description: Befolkningen (ultimo november) efter oprindelsesland, socioøkonomisk status, alder, køn og tid
measure: indhold (unit Antal)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- socio: values [TOT=I alt, 05=Selvstændige, 10=Medarbejdende ægtefæller, 15=Lønmodtager med ledelsesarbejde, 20=Lønmodtagere på højeste niveau ... 156=Kursister, 158=Produktionsskoleelever, 160=Udenlandske studerende, 170=Børn og unge (ikke under uddannelse), 175=Øvrige uden for arbejdsstyrken]
- alder: values [-15=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01

notes:
- oprindland mixes aggregate numeric codes (TOT, 30, 32, 34, 36, 38) with individual country alpha codes (PAK, SRI, SYR, etc.). Do not mix levels in the same SUM — aggregates already include country-level rows.
- socio has 30+ detailed categories. TOT=I alt is available. Use ColumnValues("ras205", "socio") to browse all values.
- kon has NO total row (only M and K). SUM across M+K to get both genders combined.
- Compare with ras208: same dimensions but ras208 has simplified socio (3 categories) and single-year ages; ras205 has detailed socio and 5-year age groups. Use ras205 for socioeconomic detail, ras208 for fine age analysis.
