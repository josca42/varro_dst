table: fact.ras204
description: Erhvervs- og beskæftigelsesfrekvenser (ultimo november) efter oprindelsesland, frekvens, alder, køn og tid
measure: indhold (unit Pct.)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- beregning: values [BFK=Beskæftigelsesfrekvens, EFK=Erhvervsfrekvens]
- alder: values [16-64=16-64 år, 16-65=16-65 år (2020 -), 1666=16-66 år (2022 -), 16-17=16-17 år, 18-19=18-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01

notes:
- beregning is a measurement selector (BFK=Beskæftigelsesfrekvens, EFK=Erhvervsfrekvens). Every row is duplicated once per beregning. Always filter to one: WHERE beregning = 'BFK'.
- oprindland mixes aggregate numeric codes (30=Dansk, 32=Indvandrere vestlige, 34=Indvandrere ikke-vestlige, 36=Efterkommere vestlige, 38=Efterkommere ikke-vestlige, TOT=I alt) with individual country alpha codes (PAK, SRI, SYR, etc.). Do not sum these together — aggregates already include their sub-countries.
- alder uses broad range codes: 16-64/16-65/1666 are the widest total-range codes (shifts over time as retirement age extended). No single "total age" code — use the widest applicable range for the year.
- No regional breakdown. Use ras200 for region-level frequency data.
