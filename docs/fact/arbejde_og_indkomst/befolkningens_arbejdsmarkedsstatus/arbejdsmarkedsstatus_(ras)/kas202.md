table: fact.kas202
description: Befolkningen efter socioøkonomisk status, alder, køn, periode og tid
measure: indhold (unit Antal)
columns:
- socio: values [TOT=I alt, 05=Selvstændige, 10=Medarbejdende ægtefæller, 15=Lønmodtager med ledelsesarbejde, 20=Lønmodtagere på højeste niveau ... 156=Kursister, 158=Produktionsskoleelever, 160=Udenlandske studerende, 170=Børn og unge (ikke under uddannelse), 175=Øvrige uden for arbejdsstyrken]
- alder: values [TOT=Alder i alt, U15=Under 15 år, 15=15 år, 16=16 år, 17=17 år ... 67=67 år, 68=68 år, 69=69 år, 70=70 år, 71-=71 og derover]
- koen: values [TOT=I alt, M=Mænd, K=Kvinder]
- tiden: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2017-01-01 to 2023-01-01

notes:
- Gender column is named koen (not kon) — same as ras202. Values: TOT=I alt, M=Mænd, K=Kvinder.
- Time-granularity selector column is named tiden (not periode) — contains same values as kas201.periode: 1=Hele året, K1–K4=quarters. Always filter: WHERE tiden = '1' for annual data. Failing to filter inflates results 5×.
- socio has 30+ detailed categories. TOT=I alt available. Use ColumnValues("kas202", "socio") to browse.
- alder has single-year ages plus TOT=Alder i alt — same structure as ras202. Best for fine age-group analysis.
- Quarterly version of ras202; adds tiden. No regional breakdown.
