table: fact.ras202
description: Befolkningen (ultimo november) efter socioøkonomisk status, alder, køn og tid
measure: indhold (unit Antal)
columns:
- socio: values [TOT=I alt, 05=Selvstændige, 10=Medarbejdende ægtefæller, 15=Lønmodtager med ledelsesarbejde, 20=Lønmodtagere på højeste niveau ... 156=Kursister, 158=Produktionsskoleelever, 160=Udenlandske studerende, 170=Børn og unge (ikke under uddannelse), 175=Øvrige uden for arbejdsstyrken]
- alder: values [TOT=Alder i alt, U15=Under 15 år, 15=15 år, 16=16 år, 17=17 år ... 67=67 år, 68=68 år, 69=69 år, 70=70 år, 71-=71 og derover]
- koen: values [TOT=I alt, M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01

notes:
- Gender column is named koen (not kon). Values: TOT=I alt, M=Mænd, K=Kvinder.
- socio has 30+ detailed categories covering the full socioeconomic ladder (selvstændige, ledelse, lønmodtagere på various levels, students, retired, etc.). Use ColumnValues("ras202", "socio") to browse all. TOT=I alt is available.
- alder has individual single-year ages (U15, 15, 16, 17 ... 70, 71-) plus TOT=Alder i alt. This enables arbitrary age-range aggregation via SQL CASE or BETWEEN-style filtering, but watch out: summing all single-year ages will double-count relative to TOT.
- No regional breakdown in this table — national level only. Use ras201 or ras209 for regional data.
