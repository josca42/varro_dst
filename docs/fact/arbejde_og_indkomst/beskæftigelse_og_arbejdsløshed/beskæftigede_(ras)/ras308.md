table: fact.ras308
description: Beskæftigede (ultimo november) efter oprindelsesland, branche (DB07), socioøkonomisk status, alder, køn og tid
measure: indhold (unit Antal)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- branche07: join dim.db on branche07=kode::text
- socio: join dim.socio on socio=kode
- alder: values [-15=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/db.md, /dim/socio.md
notes:
- branche07 uses DB07-37-group letter codes (A, B, CA...) — does NOT join dim.db. Use ColumnValues("ras308", "branche07") for labels.
- socio does NOT join dim.socio — RAS codes 5, 10, 15, 20, 25, 30, 35, 40. No total row.
- oprindland has TOT (I alt), broad groups (30=Dansk oprindelse, 32/34=Indvandrere, 36/38=Efterkommere), and many individual country codes. Use ColumnValues("ras308", "oprindland") for the full list.
- kon has only M/K — no total.
