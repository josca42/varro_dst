table: fact.kas307
description: Gennemsnitligt antal lønmodtagere efter branche (DB07), sektor, køn, periode og tid
measure: indhold (unit Antal)
columns:
- branche07: join dim.db on branche07=kode::text
- sektor: values [1000=Sektorer i alt, 1015=Stat, 1020=Regioner, 1025=Kommuner, 1030=Sociale kasser og fonde, 1035=Offentlige virksomheder, 1040=Private virksomheder, 1045=Private nonprofit-organisationer, 1050=Uoplyst]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/db.md
notes:
- periode is a critical selector column: 1=Hele året (annual average), K1-K4=quarterly averages. Always filter to one value. Annual average is NOT the sum of quarterly averages — each is an independent average for that period. For most questions, use periode='1' for the annual figure.
- branche07 uses DB07-37-group letter codes (A, B, CA...) — does NOT join dim.db. All codes at the same level, no hierarchy. Use ColumnValues("kas307", "branche07") for labels.
- sektor has a total code here: 1000=Sektorer i alt, plus 1015-1050. Does NOT join dim.esr_sekt.
- kon has TOT, M, K.
