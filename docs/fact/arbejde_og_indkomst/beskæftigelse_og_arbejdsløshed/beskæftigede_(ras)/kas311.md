table: fact.kas311
description: Gennemsnitligt antal beskæftigede efter branche (DB07), herkomst, køn, periode og tid
measure: indhold (unit Antal)
columns:
- branche07: join dim.db on branche07=kode::text; levels [2, 3, 5]
- herkomst: join dim.herkomst on herkomst=kode::text; levels [1]
- kon: values [TOT=I alt, 1=Mænd, 2=Kvinder]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/db.md, /dim/herkomst.md
notes:
- periode is a critical selector column: 1=Hele året (annual average), K1-K4=quarterly averages. Always filter to one value. Annual average is NOT the sum of quarterly averages — each is an independent average for that period. For most questions, use periode='1' for the annual figure.
- branche07 uses DB07-127 coding with two hierarchy levels: single/double-digit sector totals, 5-digit sub-sector codes. Never mix both levels. Use ColumnValues for labels. Use ColumnValues("kas311", "branche07").
- herkomst does NOT join dim.herkomst — inline codes: TOT=I alt, 1=Dansk oprindelse, 24=Indvandrere fra vestlige, 25=Indvandrere fra ikke-vestlige, 34=Efterkommere fra vestlige, 35=Efterkommere fra ikke-vestlige.
- kon uses unusual numeric coding: TOT=I alt, 1=Mænd, 2=Kvinder (same unusual coding as ras311, not the standard M/K).
