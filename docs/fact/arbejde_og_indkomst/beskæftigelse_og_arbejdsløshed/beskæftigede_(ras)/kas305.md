table: fact.kas305
description: Gennemsnitligt antal lønmodtagere efter arbejdsstedsområde, sektor, alder, køn, periode og tid
measure: indhold (unit Antal)
columns:
- arbejdssted: join dim.nuts on arbejdssted=kode; levels [1, 2, 3]
- sektor: join dim.esr_sekt on sektor::text=kode [approx]
- alder: values [TOT=Alder i alt, -16=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/esr_sekt.md, /dim/nuts.md
notes:
- periode is a critical selector column: 1=Hele året (annual average), K1-K4=quarterly averages. Always filter to one value. Annual average is NOT the sum of quarterly averages — each is an independent average for that period. For most questions, use periode='1' for the annual figure.
- arbejdssted joins dim.nuts at 98%. Code 0=Hele landet and 950=Uden for Danmark are aggregates not in dim.nuts.
- sektor does NOT join dim.esr_sekt — inline codes 1015-1050. See ras305 notes for labels.. kas305 adds a '1000=Sektorer i alt' total compared to ras305.
- alder has TOT plus age groups.
- kon has TOT, M, K.
- Measures gennemsnitligt antal lønmodtagere (average count) by workplace area, not residence.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on arbejdssted=dim_kode. Exclude arbejdssted IN (0, 950).
