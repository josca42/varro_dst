table: fact.ras305
description: Beskæftigede lønmodtagere (ultimo november) efter arbejdsstedsområde, sektor, arbejdstidens omfang, alder, køn og tid
measure: indhold (unit Antal)
columns:
- arbejdssted: join dim.nuts on arbejdssted=kode; levels [1, 2, 3]
- sektor: join dim.esr_sekt on sektor::text=kode [approx]
- arbomfang: values [H=Heltid, D=Deltid]
- alder: values [-15=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/esr_sekt.md, /dim/nuts.md
notes:
- arbejdssted joins dim.nuts at 98%. Code 0=Hele landet and 950=Uden for Danmark are aggregate codes not in dim.nuts. Use ColumnValues("nuts", "titel", for_table="ras305") to see available regions.
- sektor does NOT join dim.esr_sekt — uses inline codes 1015=Stat, 1020=Regioner, 1025=Kommuner, 1030=Sociale kasser og fonde, 1035=Offentlige virksomheder, 1040=Private virksomheder, 1045=Private nonprofit-organisationer, 1050=Uoplyst. No total code — sum all to get total lønmodtagere.
- arbomfang has only H (Heltid) and D (Deltid) — no total. Sum H+D for totals.
- kon has only M and K — no total. Sum M+K for totals.
- This table records workplace location (arbejdssted), not residence — counts people by where they work, not where they live.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on arbejdssted=dim_kode. Exclude arbejdssted IN (0, 950).
