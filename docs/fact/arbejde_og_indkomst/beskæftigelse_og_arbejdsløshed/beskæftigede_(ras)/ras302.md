table: fact.ras302
description: Beskæftigede (ultimo november) efter område, branche (DB07), socioøkonomisk status, alder, køn og tid
measure: indhold (unit Antal)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- branche07: join dim.db on branche07=kode::text
- socio: join dim.socio on socio=kode
- alder: values [-15=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/db.md, /dim/nuts.md, /dim/socio.md
notes:
- Nearly identical to ras301 but uses column name kon (not koen). Both have only M/K — no total for gender.
- omrade joins dim.nuts at 99%. Code 0=Hele landet is aggregate, not in dim.nuts. niveau 1=5 regioner, niveau 3=99 kommuner.
- branche07 uses DB07-37-group letter codes (A, B, CA...) — does NOT join dim.db. Use ColumnValues("ras302", "branche07").
- socio does NOT join dim.socio — RAS codes 5, 10, 15, 20, 25, 30, 35, 40. No total row.
- alder has no total row; 13 age groups.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
