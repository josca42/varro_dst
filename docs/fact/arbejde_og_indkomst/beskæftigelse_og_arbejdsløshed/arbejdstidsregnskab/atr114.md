table: fact.atr114
description: Arbejdstidsregnskab (kvartal) (sæsonkorrigeret) efter sektor, type, socioøkonomisk status og tid
measure: indhold (unit -)
columns:
- sektor: values [TOT=I alt, 0=Virksomheder og organisationer, 1=Offentlig forvaltning og service]
- type: values [HACS=Præsteret tid (1000 timer), ABESK=Beskæftigede (antal), AJOB=Job (antal), WSUM=Lønsum (mio. kr.)]
- socio: join dim.socio on socio=kode::text [approx]; levels [2]
- tid: date range 2008-01-01 to 2025-04-01
dim docs: /dim/socio.md

notes:
- Seasonally adjusted (sæsonkorrigeret) version with only sektor and socio breakdowns — no branche column.
- `type` is a measurement selector: HACS, ABESK, AJOB, WSUM. Always filter to one type to avoid 4x overcounting.
- `socio` has custom code `14` (not in dim.socio) and `TOT`. Only code `13` (Lønmodtagere) joins dim.socio at niveau 2.
- Simplest table in this group — use when you need seasonally adjusted totals by sektor without industry breakdown.
