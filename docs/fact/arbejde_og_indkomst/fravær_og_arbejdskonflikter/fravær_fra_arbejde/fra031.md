table: fact.fra031
description: Fravær i forbindelse med egen sygdom efter sektor, køn, fraværsindikator, periodelængde og tid
measure: indhold (unit -)
columns:
- sektor: values [1000=Sektorer i alt, 1032=Offentlig forvaltning og service, 1016=Stat (inklusiv sociale kasser og fonde), 1018=Kommuner og regioner i alt, 1020=Regioner, 1025=Kommuner, 1046=Virksomheder og organisationer, 1047=Virksomheder og organisationer - fastlønnede, 1048=Virksomheder og organisationer - timelønnede]
- kon: values [0=I alt, 1=Mænd, 2=Kvinder]
- fravaer1: values [30=Fraværsperioder (antal), 35=Fordeling af fraværsperioder (procent), 20=Fraværsdagsværk (antal), 25=Fordeling af fraværsdagsværk (procent)]
- periodelg: values [TOT=Dage i alt, 100=1 dag, 105=2 dage, 110=3-7 dage, 115=8-14 dage, 120=15-21 dage, 125=22-30 dage, 130=31-60 dage, 135=61-90 dage, 140=91-120 dage, 145=Mere end 120 dage, 150=Uafsluttet periodelængde]
- tid: date range 2013-01-01 to 2023-01-01
notes:
- This table covers own sickness (Eigen sygdom) only — no fravaer cause column.
- fravaer1 has 4 values here (not 8 like in other fra0XX tables): 30=Fraværsperioder antal, 35=Fordeling af fraværsperioder (%), 20=Fraværsdagsværk antal, 25=Fordeling af fraværsdagsværk (%). The % values (25, 35) are distribution shares that sum to 100 across periodelg categories — do NOT sum indhold across periodelg when using these.
- periodelg = spell duration categories (TOT=I alt, 1 day, 2 days, 3-7 days, ..., >120 days, and 150=Uafsluttet periodelændde for ongoing spells). TOT is the total across all durations.
- sektor='1000' and kon='0' are totals. Filter sektor to avoid double-counting sector codes alongside the total.
