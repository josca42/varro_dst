table: fact.fra029
description: Fravær på arbejdspladser i virksomheder og organisationer efter branche (DB07), arbejdsfunktion, køn, størrelsesgruppe, fraværsindikator, fraværsårsag og tid
measure: indhold (unit -)
columns:
- branche07: join dim.db on branche07=kode::text [approx]
- arbfunk: join dim.disco on arbfunk=kode::text [approx]; levels [1, 2]
- kon: values [0=I alt, 1=Mænd, 2=Kvinder]
- strgrupipr: values [0=Ansatte i alt, 11=10-50 fuldtidsansatte, 22=Mere end 50 fuldtidsansatte]
- fravaer1: values [10=Fraværsprocent, 40=Gennemsnitlige antal fraværsdagsværk pr. fuldtidsansat, 50=Gennemsnitlige antal fraværsperioder pr. helårsansat, 60=Gennemsnitlige antal kalenderdage pr. fraværsperiode, 70=Fuldtidsansatte, 80=Helårsansatte, 30=Fraværsperioder (antal), 20=Fraværsdagsværk (antal)]
- fravaer: values [1100=Egen sygdom, 1200=Børns sygdom, 1300=Arbejdsulykke, 1400=Barsels- og adoptionsorlov]
- tid: date range 2013-01-01 to 2023-01-01
dim docs: /dim/db.md, /dim/disco.md
notes:
- This table covers virksomheder og organisationer (private/org sector) only.
- branche07 uses letter-based DB07 section codes (B, CA, CB, CC, ..., S, X, TOT) — these do NOT join to dim.db. Use ColumnValues("fra029", "branche07") to get labels (e.g. 'CA'='CA Føde-, drikke- og tobaksvareindustri'). 37 codes total.
- arbfunk joins dim.disco at niveau 1 (single-digit major groups: 1-9) and niveau 2 (two-digit sub-groups). 'TOT'=I alt and '9999'=Uoplyst do not join. Mixing niveaus causes double-counting — filter to one niveau.
- strgrupipr is a size-group selector (0=Ansatte i alt, 11=10-50 fuldtidsansatte, 22=Mere end 50). Filter to '0' for totals or split by size.
- fravaer1 is a measurement-type selector (8 values) — always filter to one value.
- fravaer is absence cause (4 values). kon='0' is the total.
