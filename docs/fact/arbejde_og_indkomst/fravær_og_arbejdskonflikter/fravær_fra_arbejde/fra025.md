table: fact.fra025
description: Fravær på statslige arbejdspladser efter område, køn, fraværsårsag, branche (DB07), fraværsindikator og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1]
- kon: values [0=I alt, 1=Mænd, 2=Kvinder]
- fravaer: values [1100=Egen sygdom, 1200=Børns sygdom, 1300=Arbejdsulykke, 1400=Barsels- og adoptionsorlov]
- branche07: join dim.db on branche07=kode::text [approx]; levels [5]
- fravaer1: values [10=Fraværsprocent, 40=Gennemsnitlige antal fraværsdagsværk pr. fuldtidsansat, 50=Gennemsnitlige antal fraværsperioder pr. helårsansat, 60=Gennemsnitlige antal kalenderdage pr. fraværsperiode, 70=Fuldtidsansatte, 80=Helårsansatte, 30=Fraværsperioder (antal), 20=Fraværsdagsværk (antal)]
- tid: date range 2013-01-01 to 2023-01-01
dim docs: /dim/db.md, /dim/nuts.md
notes:
- This table covers statslige (state) workplaces only — no sektor column needed.
- branche07 uses 11 special 5-digit state-sector codes (52000=Hjælpevirksomhed til transport, 60000=Radio- og tv-stationer, 71000=Arkitekter og rådgivende ingeniører, 72000=Forskning og udvikling, 84001=Offentlig administration, 84002=Forsvar og politi, 85002=Gymnasier og erhvervsfaglige skoler, 85003=Videregående uddannelsesinstitutioner, 85004=Voksenundervisning, 91000=Biblioteker og museer, 94000=Organisationer og foreninger) plus TOT. These do NOT join to dim.db — the codes are state-specific. Use ColumnValues("fra025", "branche07") for labels.
- omrade joins dim.nuts at niveau 1 (5 regions). Code '0'=Hele landet not in dim.nuts.
- fravaer1 is a measurement-type selector (8 values) — always filter to one value.
- fravaer is absence cause (4 values). kon='0' is the total.
- Map: /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade='0'.
