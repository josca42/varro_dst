## Brug

Simpel join:

```sql
SELECT f.indhold, d.titel
FROM fact.<fact_tabel> f
JOIN dim.db_10 d ON f.db_10 = d.kode
WHERE d.niveau = 1
```

Aggregering via parent_kode (fra fineste til groveste niveau):

```sql
SELECT top.titel, SUM(f.indhold)
FROM fact.<fact_tabel> f
  JOIN dim.db_10 detail ON f.db_10 = detail.kode AND detail.niveau = 2
  JOIN dim.db_10 top ON top.kode = detail.parent_kode AND top.niveau = 1
GROUP BY top.titel
```
