notes:
- **Sector-level economics (joek1/joek2/joek3)**: joek1 has the full national accounts for the agricultural sector (production, costs, income) in a hierarchical produkt tree; joek2 adds a regional dimension (5 regions) but only covers 2022–2023. joek3 covers only labor (årsværk). All three require filtering prisenhed/region carefully. Very short series (3 years max).

- **Farm account results — all vs. heltid vs. deltid**: jord1 (alle bedrifter), jord2 (heltidsbedrifter classified by årsværk/type), jord3 (deltidsbedrifter). All share the same regnskposter accounting tree (~234 items, mixed units) and kvartil dimension. jord1 and jord6 also include regional and standardoutput breakdowns via the bedriftstand column. Counterpart key-ratio tables: jord6 (alle), jord7 (heltid), jord8 (deltid). jord9 covers energy consumption for heltidsbedrifter. jord4/jord5 cover family economy (not farm business).

- **Historical benchmark series**: jord100 goes back to 1916 — the only table here with a century-scale series. Only 7 accounting items, but per-ha and per-farm splits. Use for long-run trend questions.

- **Production branch economics (pgkonv/pgoeko)**: Four tables split by konventionel vs. økologisk and by afgrøder (crops: pgkonv1/pgoeko1) vs. husdyr (livestock: pgkonv2/pgoeko2). Each has many regnskabsposter with different units per branch. Only 3 years (2021–2023).

- **Animal product indices (ani301/302/303)**: Same 2015=100 index data at three time granularities — annual (301), quarterly (302), monthly (303). More products available at annual frequency (11 varegr) than monthly (5). The indekstype column is always a selector (mængde vs. pris) — filter to ONE.

- **Prices — sale prices**: lpris10 (monthly, from 2005, broadest product list including mink/poultry), lpris32 (annual, 2020–2024, adds organic egg categories). Both have an enhed selector (løbende priser vs. % change) — filter to enhed=320 for actual prices.

- **Prices — input/factor prices**: lpris38 (annual, 2020–2024) covers seeds, fertilizer, pesticides, energy, labor, and land costs. Same enhed selector pattern.

- **Price indices — terms of trade**: lpris22 (quarterly, through 2025Q1) and lpris28 (annual, through 2024) both give the same produkt hierarchy of sale/purchase price indices (2020=100). Code 0=Jordbrugets bytteforhold is the terms-of-trade ratio. Both have a tal selector — filter to ONE.

- **Land prices by region**: lpris37 is the only table with regional land price data (5 regions, 2015–2024): forpagtningsafgift and landbrugsjord købspris. Join region to dim.nuts (both smallint, niveau=1).

- **Direct subsidies**: tilskud2 covers 2021–2024, no regional breakdown. Only table for subsidy breakdown by type.

- **Historical grain levy rates**: kapit1 (kapitelstakster, 1985–2024) uses DST-specific levy districts, not standard regions. Byg and Hvede only.

- **Common pitfall across all tables**: Every table with an enhed, prisenhed, tal, belob, or indekstype column is a measurement selector — every row combination appears N times (once per selector value). Always filter to ONE selector value before aggregating.

- **Suffix pattern**: lpris tables use a numeric suffix that does NOT encode time granularity consistently (unlike folk1a/folk1am pattern). lpris10=monthly sale prices, lpris22=quarterly index, lpris28=annual index, lpris32=annual sale prices, lpris37=annual land prices, lpris38=annual input prices. Read the description carefully.

- **Map**: joek2 and lpris37 support choropleth maps at region level (5 Danish regions) via /geo/regioner.parquet — merge on region=dim_kode, exclude region=0.
