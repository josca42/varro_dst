table: fact.kas300
description: Gennemsnitligt antal beskæftigede efter branche (DB07), socioøkonomisk status, alder, køn, periode og tid
measure: indhold (unit Antal)
columns:
- branche07: join dim.db on branche07=kode::text [approx]; levels [2, 3, 5]
- socio: join dim.socio on socio=kode; levels [1]
- alder: values [TOT=Alder i alt, -16=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2008-01-01 to 2023-01-01
dim docs: /dim/db.md, /dim/socio.md
notes:
- periode is a critical selector column: 1=Hele året (annual average), K1-K4=quarterly averages. Always filter to one value. Annual average is NOT the sum of quarterly averages — each is an independent average for that period. For most questions, use periode='1' for the annual figure.
- branche07 uses DB07-127 coding with two hierarchy levels: single/double-digit sector totals, 5-digit sub-sector codes. Never mix both levels. Use ColumnValues for labels. Use ColumnValues("kas300", "branche07").
- socio does NOT join dim.socio — RAS codes 5, 10, 15, 20, 25, 30, 35, 40 (employed persons only). See ras300 notes for labels. No total row — sum all 8 codes.
- alder has TOT plus age groups (note: -16 code for under 16, not -15 as in ras300).
- kon has TOT, M, K.
- Measures gennemsnitligt antal beskæftigede (average count over the period) — not a snapshot like ras300.
