table: fact.atr122
description: Arbejdstidsregnskab (år), indeks efter køn, sektor, type og tid
measure: indhold (unit Indeks)
columns:
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- sektor: values [TOT=I alt, 0=Virksomheder og organisationer, 1=Offentlig forvaltning og service]
- type: values [HAC=Præsteret tid (1000 timer), ABESK=Beskæftigede (antal)]
- tid: date range 2008-01-01 to 2023-01-01

notes:
- `indhold` is an index value (base=2008), NOT counts or hours. Do not sum across rows — compare values over time or across categories.
- `type` has only 2 values here: HAC (præsteret tid) and ABESK (beskæftigede). Always filter to one.
- No branche column — only kon, sektor, type, tid. Use atr116 if industry breakdown is needed.
- Useful for tracking relative change since 2008 in hours worked or employment, split by gender and sector.
