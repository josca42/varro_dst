table: fact.kas200
description: Erhvervs- og beskæftigelsesfrekvenser efter område, herkomst, alder, køn, frekvens, periode og tid
measure: indhold (unit Pct.)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- herkomst: values [0=I alt, 10=Personer med dansk oprindelse, 20=Indvandrere, 24=Indvandrere fra vestlige lande, 25=Indvandrere fra ikke-vestlige lande, 30=Efterkommere, 34=Efterkommere fra vestlige lande, 35=Efterkommere fra ikke-vestlige lande]
- alder: values [TOT=Alder i alt, 16-17=16-17 år, 18-19=18-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- beregning: values [BFK=Beskæftigelsesfrekvens, EFK=Erhvervsfrekvens]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2017-01-01 to 2023-01-01
dim docs: /dim/nuts.md

notes:
- periode is a time-granularity selector (1=Hele året, K1–K4=quarters). Rows for each quarter AND for the annual total exist in the same table. Always filter to one: WHERE periode = '1' for annual data, or a specific quarter. Failing to filter inflates results 5×.
- beregning is also a measurement selector (BFK/EFK) — doubles rows. Filter to one: WHERE beregning = 'BFK'.
- omrade joins dim.nuts on omrade=kode. dim.nuts has 5 regioner (niveau 1), 11 landsdele (niveau 2), 99 kommuner (niveau 3). Filter WHERE d.niveau = N.
- omrade='0' is the national total — not in dim.nuts. Use directly: WHERE f.omrade = '0'.
- KAS = quarterly version of RAS. kas200 corresponds to ras200 but adds periode column and covers from 2017. Use ras200 for data before 2017, kas200 for quarterly granularity from 2017.
- Sample query — BFK by region, annual 2022: SELECT f.omrade, d.titel, f.indhold FROM fact.kas200 f JOIN dim.nuts d ON f.omrade = d.kode WHERE d.niveau = 1 AND f.herkomst = '0' AND f.alder = 'TOT' AND f.kon = 'TOT' AND f.beregning = 'BFK' AND f.periode = '1' AND f.tid = '2022-01-01';
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
