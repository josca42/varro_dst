table: fact.kas204
description: Erhvervs- og beskæftigelsesfrekvenser efter oprindelsesland, frekvens, alder, køn, periode og tid
measure: indhold (unit Pct.)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- beregning: values [BFK=Beskæftigelsesfrekvens, EFK=Erhvervsfrekvens]
- alder: values [TOT=Alder i alt, 16-17=16-17 år, 18-19=18-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- tiden: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2017-01-01 to 2023-01-01

notes:
- beregning is a measurement selector (BFK/EFK) — doubles rows. Filter to one: WHERE beregning = 'BFK'.
- Time-granularity selector is named tiden (not periode): 1=Hele året, K1–K4=quarters. Always filter to one: WHERE tiden = '1'. Failing to filter inflates results 5×.
- oprindland mixes aggregate codes and individual country codes — same as ras204. Do not sum aggregate and country-level codes together.
- alder has TOT=Alder i alt (unlike ras204's range codes). Quarterly version of ras204 from 2017.
