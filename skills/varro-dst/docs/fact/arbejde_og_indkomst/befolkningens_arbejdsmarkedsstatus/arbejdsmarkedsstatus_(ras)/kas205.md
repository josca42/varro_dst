table: fact.kas205
description: Befolkningen efter oprindelsesland, socioøkonomisk status, alder, køn, periode og tid
measure: indhold (unit Antal)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- socio: values [TOT=I alt, 05=Selvstændige, 10=Medarbejdende ægtefæller, 15=Lønmodtager med ledelsesarbejde, 20=Lønmodtagere på højeste niveau ... 156=Kursister, 158=Produktionsskoleelever, 160=Udenlandske studerende, 170=Børn og unge (ikke under uddannelse), 175=Øvrige uden for arbejdsstyrken]
- alder: values [TOT=Alder i alt, -16=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2017-01-01 to 2023-01-01

notes:
- periode is a time-granularity selector (1=Hele året, K1–K4=quarters). Always filter to one: WHERE periode = '1'. Failing to filter inflates results 5×.
- oprindland mixes aggregate codes and individual country codes — same as ras205. Do not sum aggregate and country-level codes together.
- socio has 30+ detailed categories. TOT=I alt available.
- kon has TOT=I alt (unlike ras205 which has no total).
- alder has TOT=Alder i alt plus 5-year age groups. Quarterly version of ras205 from 2017. Compare with kas208: kas208 = simplified socio + single-year ages.
