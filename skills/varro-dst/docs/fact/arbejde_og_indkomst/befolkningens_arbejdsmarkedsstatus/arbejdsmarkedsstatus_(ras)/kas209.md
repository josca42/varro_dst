table: fact.kas209
description: Befolkningen efter område, uddannelse, socioøkonomisk status, alder, køn, periode og tid
measure: indhold (unit Antal)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- uddannelse: values [TOT=I alt, H10=H10 Grundskole, H20=H20 Gymnasiale uddannelser, H30=H30 Erhvervsfaglige uddannelser, H35=H35 Adgangsgivende uddannelsesforløb, H40=H40 Korte videregående uddannelser, KVU, H50=H50 Mellemlange videregående uddannelser, MVU, H60=H60 Bacheloruddannelser, BACH, H70=H70 Lange videregående uddannelser, LVU, H80=H80 Ph.d. og forskeruddannelser, H90=H90 Uoplyst mv.]
- socio: values [TOT=I alt, 05=Selvstændige, 10=Medarbejdende ægtefæller, 15=Lønmodtager med ledelsesarbejde, 20=Lønmodtagere på højeste niveau ... 156=Kursister, 158=Produktionsskoleelever, 160=Udenlandske studerende, 170=Børn og unge (ikke under uddannelse), 175=Øvrige uden for arbejdsstyrken]
- alder: values [-15=Under 16 år, 16-19=16-19 år, 20-24=20-24 år, 25-29=25-29 år, 30-34=30-34 år, 35-39=35-39 år, 40-44=40-44 år, 45-49=45-49 år, 50-54=50-54 år, 55-59=55-59 år, 60-64=60-64 år, 65-66=65-66 år, 67-=67 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- periode: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2017-01-01 to 2023-01-01
dim docs: /dim/nuts.md

notes:
- periode is a time-granularity selector (1=Hele året, K1–K4=quarters). Always filter to one: WHERE periode = '1'. Failing to filter inflates results 5×.
- omrade joins dim.nuts on omrade=kode. dim.nuts has 5 regioner (niveau 1), 11 landsdele (niveau 2), 99 kommuner (niveau 3). Filter WHERE d.niveau = N.
- omrade='0' is the national total — not in dim.nuts.
- socio has 30+ detailed categories. TOT=I alt available.
- uddannelse: 10 education levels (H10–H90) plus TOT=I alt. See ras209 notes for details.
- alder: no grand total (no -15 code is '-15' = Under 16 år). Quarterly version of ras209 from 2017.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
