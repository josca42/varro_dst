table: fact.ras208
description: Befolkningen (ultimo november) efter oprindelsesland, socioøkonomisk status, alder, køn og tid
measure: indhold (unit Antal)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- socio: values [0=Beskæftigede, 50=Arbejdsløse, 55=Uden for arbejdsstyrken]
- alder: values [TOT=Alder i alt, U15=Under 15 år, 15=15 år, 16=16 år, 17=17 år ... 67=67 år, 68=68 år, 69=69 år, 70=70 år, 71-=71 og derover]
- kon: values [M=Mænd, K=Kvinder]
- tid: date range 2008-01-01 to 2023-01-01

notes:
- oprindland mixes aggregate codes and individual country codes — same as ras205. TOT=I alt available.
- socio is simplified to 3 values (0=Beskæftigede, 50=Arbejdsløse, 55=Uden for arbejdsstyrken). No detailed socioeconomic breakdown — use ras205 for that.
- alder has single-year ages (U15, 15, 16 ... 70, 71-) plus TOT=Alder i alt. Best for fine age-specific queries; sum individual ages matches TOT.
- kon has NO total row (only M and K). SUM across M+K for combined gender.
- Compare with ras205: ras208 = simplified socio + single-year ages; ras205 = detailed socio + 5-year age groups.
