table: fact.kas208
description: Befolkningen efter oprindelsesland, socioøkonomisk status, alder, køn, periode og tid
measure: indhold (unit Antal)
columns:
- oprindland: values [TOT=I alt, 30=Dansk oprindelse, 32=Indvandrere fra vestlige lande, 34=Indvandrere fra ikke-vestlige lande, 36=Efterkommere fra vestlige lande ... PAK=Pakistan, SRI=Sri Lanka, SYR=Syrien, THA=Thailand, VIE=Vietnam]
- socio: values [TOT=I alt, 00=Beskæftigede, 50=Arbejdsløse, 55=Uden for arbejdsstyrken]
- alder: values [TOT=Alder i alt, U15=Under 15 år, 15=15 år, 16=16 år, 17=17 år ... 67=67 år, 68=68 år, 69=69 år, 70=70 år, 71-=71 og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- tiden: values [1=Hele året, K1=1. kvartal, K2=2. kvartal, K3=3. kvartal, K4=4. kvartal]
- tid: date range 2017-01-01 to 2023-01-01

notes:
- Time-granularity selector is named tiden (not periode): 1=Hele året, K1–K4=quarters. Always filter to one: WHERE tiden = '1'. Failing to filter inflates results 5×.
- oprindland mixes aggregate codes and individual country codes — same as ras208.
- socio is simplified to 3 values: TOT=I alt, 00=Beskæftigede, 50=Arbejdsløse, 55=Uden for arbejdsstyrken.
- alder has single-year ages plus TOT=Alder i alt — best for fine age analysis.
- kon has TOT=I alt (unlike ras208 which has no total).
- Quarterly version of ras208 from 2017. Compare with kas205: kas205 = detailed socio + age groups; kas208 = simplified socio + single-year ages.
