table: fact.tema9013
description: Biler og deres C02-udledning efter enhed og tid
measure: indhold (unit Indeks)
columns:
- enhed: values [445.0=Udledning fra familiernes bilkørsel (CO2), 448.0=Energieffektivitet (l/km) nyreg. biler, 455.0=Bestand af biler i familierne]
- tid: date range 1990-01-01 to 2024-01-01
notes:
- enhed encodes 3 independent index time series — never sum across enhed. Each is indexed to year 2000=100: 445=CO2-udledning fra familiernes bilkørsel, 455=Bestand af biler i familierne. The enhed=448 (Energieffektivitet l/km) appears in the column definition but its rows have null enhed and all-zero values in the database — treat that series as unavailable; use ee1 instead for fuel efficiency.
- Annual data from 1990-01-01 to 2024-01-01. The index base is 2000 (value=100 in year 2000).
- Simple structure: enhed × tid only. Filter to one enhed for a time series query.
- Use this table for a high-level overview of CO2 trends and car stock growth indexed to a common baseline. For absolute CO2 or detailed car stock numbers, use dedicated tables in other subjects.
