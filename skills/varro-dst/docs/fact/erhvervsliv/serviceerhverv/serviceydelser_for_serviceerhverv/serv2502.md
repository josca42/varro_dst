table: fact.serv2502
description: Vikarbureauer og anden personaleformidling efter serviceydelser og tid
measure: indhold (unit 1.000 kr.)
columns:
- servyd: values [TOT=I alt, 70000=Rekruttering af ledere og mellemledere, 70010=Formidling af fast arbejde undtagen rekruttering af ledere og mellemledere, 70020=It og telekommunikation, 70030=Andet kontorpersonale, 70040=Handelsbranchen, 70045=Produktion, 70055=Bygge/anlæg, 70060=Transport, lager og logistik, 70070=Hotel- og restaurationsbranchen, 70080=Sygeplejersker, 70090=Social- og sundhedsassistenter, 70100=Social- og sundhedshjælpere, 70110=Læger, 70120=Øvrigt personale inden for sygepleje og sundhed, 70130=Øvrige personalekategorier, 70135=Anden personaleformidling, 70140=Andre serviceydelser i.a.n.]
- tid: date range 2023-01-01 to 2023-01-01
notes:
- Single snapshot: only 2023 data. No time-series analysis possible.
- servyd contains 18 values including TOT (I alt), which is the aggregate of all personnel types. Exclude it when summing across types: WHERE servyd != 'TOT'.
- Note that serv2502 measures revenue (1.000 kr.) from temp agency services, while serv2511 measures the actual hours sold (timer) for similar categories. Use serv2502 for revenue breakdown by service type, serv2511 for volume (hours) by client industry.
- Use ColumnValues("serv2502", "servyd") to see all 18 service type codes with labels.
