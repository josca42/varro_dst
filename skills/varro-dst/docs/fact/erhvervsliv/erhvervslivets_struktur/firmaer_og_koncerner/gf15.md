table: fact.gf15
description: Generel firmastatistik efter branche (DB07 10-grp), virksomhedsform, enhed og tid
measure: indhold (unit -)
columns:
- branchedb0710: values [TOT=TOT Erhverv i alt, 1=1 Landbrug, skovbrug og fiskeri, 2=2 Industri, råstofindvinding og forsyningsvirksomhed, 3=3 Bygge og anlæg, 4=4 Handel og transport mv., 5=5 Information og kommunikation, 6=6 Finansiering og forsikring, 7=7 Ejendomshandel og udlejning, 8=8 Erhvervsservice, 9=9 Offentlig administration, undervisning og sundhed, 10=10 Kultur, fritid og anden service, 11=11 Uoplyst aktivitet]
- virkf1: values [TOT=I ALT, 1=Enkeltmandsfirma, 2=Interessentskab mv., 4=Aktieselskab, 6=Fond, forening mv., 3=Anpartsselskab, 5=Andelsforening, 7=Offentlig myndighed, 9=Anden ejer]
- enhed: values [AFI=Firmaer (antal), AFU=Fuldtidsansatte (antal), OMS=Omsætning (mio kr.) ]
- tid: date range 2019-01-01 to 2023-01-01

notes:
- This is the only table in the subject with virksomhedsform (legal form). virkf1 has 9 values including TOT (I ALT). Always filter to one value or use TOT for totals.
- enhed is a measurement selector (AFI=firmaer, AFU=fuldtidsansatte, OMS=omsætning). Always filter to one value — every branchedb0710+virkf1 combination is repeated 3 times.
- branchedb0710 is fully inline (TOT + 10 sector codes 1–11 with full Danish labels). No dim join — labels are embedded in the column values. Use ColumnValues("gf15", "branchedb0710") to see all.
- This table has three selector dimensions (branchedb0710, virkf1, enhed). A query without filtering all three inflates results by up to 10×11×3 = 330x.
