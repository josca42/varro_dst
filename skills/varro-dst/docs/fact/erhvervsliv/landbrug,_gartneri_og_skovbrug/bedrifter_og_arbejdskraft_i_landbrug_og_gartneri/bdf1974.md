notes:

- areal1: H05 = I alt. Use H05 for totals; smaller brackets (H10-H92) are fine-grained size classes.
- Only useful for historical context (1970-1981). Breaks in series exist between this table and bdf52/bdf11 which start from 1984/2006 and use modern geography.
