table: fact.bdf907
description: Bedrifter med andre aktiviteter end landbrug efter område, aktivitet og tid
measure: indhold (unit Antal)
columns:
- omrade: join dim.landbrugslandsdele on omrade::text=kode [approx]; levels [3]
- aktivi: values [5050=Maskinstation, 5055=Pelsdyrbrug, 5060=Landboturisme, 5065=Håndværk, 5070=Forarbejdning af landbrugsprodukter, 5075=Salg af vedvarende energi, 5080=Gårdbutikker, 5085=Rideskole, 5090=Andre sekundære aktiviteter, 5095=Alle bedrifter med sekundære aktiviteter]
- tid: date range 2007-01-01 to 2023-01-01
dim docs: /dim/landbrugslandsdele.md

notes:
- DIMENSION DOC IS WRONG: omrade does NOT join dim.landbrugslandsdele. It joins dim.nuts with plain integer codes identical to bdf11: niveau 1 (81-85), niveau 2 (4,7,8,9,10), aggregates 0 and 15. Use JOIN dim.nuts d ON f.omrade = d.kode.
- aktivi categories are NOT mutually exclusive — a single farm can have multiple secondary activities (e.g., both landboturisme and gårdbutik). Never sum across specific activity codes; that inflates counts.
- Code 5095=Alle bedrifter med sekundære aktiviteter is the only valid total for "how many farms have any secondary activity". For specific activity counts, use individual codes (5050-5090) independently.
- Map: /geo/regioner.parquet (niveau 1, omrade 81-85) or /geo/landsdele.parquet (niveau 2, omrade 4,7,8,9,10 — 5 of 11 landsdele only) — merge on omrade=dim_kode. Exclude omrade IN (0, 15).
