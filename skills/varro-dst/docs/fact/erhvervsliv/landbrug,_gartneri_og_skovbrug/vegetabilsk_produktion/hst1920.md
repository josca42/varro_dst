table: fact.hst1920
description: Høstresultat (historisk) efter afgrøde og tid
measure: indhold (unit -)
columns:
- afgrode: values [HIST005=Hvede (1000 ha), HIST010=Rug (1000 ha), HIST015=Byg (1000 ha), HIST020=Havre (1000 ha), HIST025=Andet korn (1000 ha) ... HIST140=Høst af bælgsæd (1000 ton), HIST145=Høst af foderroer (1000 ton), HIST150=Høst af sukkerroer (1000 ton), HIST155=Høst af kartofler (1000 ton), HIST160=Høst af rodfrugter i alt (1000 ton)]
- tid: date range 1920-01-01 to 1981-01-01

notes:
- The unit is embedded in each afgrode code label, e.g. HIST005=Hvede (1000 ha), HIST060=Produktion af hvede (1000 ton). There is no separate enhed column — the afgrode code determines both what is measured and in what unit.
- This means area and production for the same crop are two separate afgrode codes, not rows that can be compared directly. Select the right HIST code for your question.
- National-level only, no regional breakdown. Use hst77 (2006–) for regional detail or modern harvests.
