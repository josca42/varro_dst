table: fact.bu43
description: Børn og unge der modtager støtte pr. 31. december (andel af 0-22-årige) efter område, Foranstaltning, alder og tid
measure: indhold (unit Pct.)
columns:
- omrade: join dim.nuts on omrade=kode; levels [3]
- foranstalt: values [IALT=Udsatte børn og unge i alt, FORB=Børn og unge med støttende indsatser, ANBR=Anbragte børn og unge]
- alder: values [TOT22=I alt, 0-22 år, 9917=0-17 år, 1822=18-22 år]
- tid: date range 2015-01-01 to 2024-01-01
dim docs: /dim/nuts.md

notes:
- This is a percentage table (Pct.) — indhold is a share of 0-22-årige with support. Never sum across rows; compare directly.
- omrade joins dim.nuts at niveau 3 (kommuner, 98 kommuner). Code 0 = national total, not in dim. Use ColumnValues("nuts", "titel", for_table="bu43") to see available kommuner.
- foranstalt has only 3 values: IALT (all udsatte), FORB (støttende indsatser), ANBR (anbragte). FORB and ANBR are both subsets of IALT. Filter to exactly one.
- alder has 3 groups: TOT22=all 0-22 år, 9917=0-17 år, 1822=18-22 år. Filter to one.
- Simple query pattern: JOIN dim.nuts d ON f.omrade=d.kode WHERE foranstalt='ANBR' AND alder='TOT22' AND tid='2024-01-01' AND d.niveau=3
- Map: /geo/kommuner.parquet — merge on omrade=dim_kode. Exclude omrade=0.
