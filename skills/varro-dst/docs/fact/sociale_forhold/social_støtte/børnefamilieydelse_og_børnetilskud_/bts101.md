table: fact.bts101
description: Modtagere af børnefamilieydelser og børnetilskud efter område, tilskudsart, køn, antal børn, enhed og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode; levels [3]
- tilskudsart: values [1000=Børneydelse, 1005=Ungeydelse, 1010=Ordinært børnetilskud (til enlige), 1015=Ekstra børnetilskud (ekstra til enlige), 1020=Særligt børnetilskud (døde forældre ellere ukendt far), 1025=Flerbørnstilskud (Flerlinger under 7 år), 1030=Mildlertidig børnetilskud (familier med nedsat kontanthjælpsydelse, 2019-2023)]
- kon: values [10=Køn i alt, M=Mænd, K=Kvinder]
- antborn: values [0=Personer i alt, 1=1 barn, 2=2 børn, 3=3 børn, 41=4 eller flere børn]
- enhed: values [105=Antal personer der modtager ydelsen, 120=Gennemsnitlig udbetaling til personer med tilskudsarten (kr.)]
- tid: date range 2017-01-01 to 2024-10-01
dim docs: /dim/nuts.md

notes:
- enhed is a measurement selector: 105 = antal personer der modtager ydelsen, 120 = gennemsnitlig udbetaling (kr.). Always filter to one enhed value — mixing both silently doubles row counts.
- omrade joins dim.nuts at niveau 3 only (98 kommuner). Use ColumnValues("nuts", "titel", for_table="bts101") to see available kommuner. omrade=0 is the national total and omrade=997 is "uoplyst" (unassignable to a kommune) — neither exists in dim.nuts. Exclude them when aggregating across kommuner (997538 = 988981 kommuner + 8557 uoplyst for Børneydelse 2024-Q4).
- To get a clean national count, use omrade='0' directly rather than summing kommuner.
- 5 dimension columns require filtering to avoid overcounting: to get recipients by tilskudsart, also set kon='10' (i alt) and antborn='0' (i alt).
- tilskudsart 1000 (Børneydelse) is the main universal benefit. The others (1010, 1015, 1020, 1025, 1030) are targeted supplements mostly for enlige or special cases. tilskudsart 1030 (midlertidig) only existed 2019–2023.
- kon codes: 10=i alt, M=mænd, K=kvinder — note that 10 is the string "10", not "TOT".
- Map: /geo/kommuner.parquet — merge on omrade=dim_kode. Exclude omrade IN (0, 997).
