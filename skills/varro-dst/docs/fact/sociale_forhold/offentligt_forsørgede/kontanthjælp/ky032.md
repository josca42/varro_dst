table: fact.ky032
description: Kontanthjælp (personer og beløb) efter enhed, ydelsestype, alder, køn og tid
measure: indhold (unit -)
columns:
- tal: values [1=Berørte personer (antal), 2=Helårsmodtagere (antal), 3=Udbetalt beløb (1.000 kr.)]
- ydelsestype: values [10000=Ydelser i alt, 10050=Kontanthjælp i alt, 10025=Uddannelseshjælp i alt, 10060=Løntilskud vedrørende kontant- og uddannelseshjælpsmodtagere m. fl. I alt, 10070=Kontant- og uddannelseshjælp under forrevalidering i alt, 10020=Kontanthjælp og integrationsydelse til udlændinge m.fl. i alt, 10040=Revalideringsydelse i alt, 10032=Kontantydelse i alt (oktober 2015 -), 10035=Arbejdsmarkedsydelse i alt (januar 2014 -), 10030=Særlig uddannelsesydelse i alt ( -januar 2014)]
- alder: values [TOT=Alder i alt, 16-24=16-24 år, 25-29=25-29 år, 30-39=30-39 år, 40-49=40-49 år, 50-59=50-59 år, 6099=60 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- tid: date range 2007-01-01 to 2024-01-01
notes:
- tal is a measurement selector with 3 values: 1=Berørte personer (antal), 2=Helårsmodtagere (antal), 3=Udbetalt beløb (1.000 kr.). Always filter to one value.
- Helårsmodtagere (tal=2) is an annual FTE measure, not a headcount.
- ydelsestype is hierarchical — 10000=Ydelser i alt covers all sub-categories. Do not sum all codes.
- Data ends at 2024-01-01. For current data use ky12 (persons only) or ky13.
- All dimension columns have TOT rows. Filter ydelsestype, alder, and kon to avoid overcounting.
