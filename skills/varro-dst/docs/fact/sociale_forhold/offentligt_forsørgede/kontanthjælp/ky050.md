table: fact.ky050
description: Hjælp i særlige tilfælde (personer og beløb) efter enhed, ydelsestype, alder, køn og tid
measure: indhold (unit -)
columns:
- tal: values [1=Berørte personer (antal), 2=Helårsmodtagere (antal), 3=Udbetalt beløb (1.000 kr.)]
- ydelsestype: values [KST=HJÆLP I SÆRLIGE TILFÆLDE I ALT, 572004=Hjælp vedr. samværsret med børn, 572006=Hjælp til sygebehandling, 572008=Hjælp til enkeltudgifter, 572007=Tilskud til tandpleje til økonomisk vanskeligt stillede, 572009=Merudgiftsydelse vedr. børn med nedsat funktionsevne, 572010=Merudgifter for voksne med nedsat funktionsevne, 572015=Tabt arbejdsfortjeneste ved forsørgelse af børn med nedsat funktionsevne, 572016=Udgifter ved køb af træningsredskaber, kurser, hjælpere mv. i forbindelse med hjemmetræning, 572017=Efterlevelseshjælp, 572018=Hjælp til udsatte lejere, 573017=Løbende hjælp til visse persongrupper, fængslede mv., 575012=Hjælp til hjælpemidler mv. til aktiverede]
- alder: values [TOT=Alder i alt, 0-15=0-15 år, 16-24=16-24 år, 25-29=25-29 år, 30-39=30-39 år, 40-49=40-49 år, 50-59=50-59 år, 6099=60 år og derover]
- kon: values [TOT=I alt, M=Mænd, K=Kvinder]
- tid: date range 2007-01-01 to 2024-01-01
notes:
- This table covers "hjælp i særlige tilfælde" — a separate benefit type from regular kontanthjælp. Subject matter is distinct: one-off payments for specific needs (dental, equipment, childcare costs, etc.).
- tal is a measurement selector with 3 values: 1=Berørte personer, 2=Helårsmodtagere, 3=Udbetalt beløb (1.000 kr.). Always filter to one value.
- ydelsestype codes use KST prefix and 572xxx series — completely different from regular kontanthjælp codes (10xxx). KST=HJÆLP I SÆRLIGE TILFÆLDE I ALT is the total.
- alder includes 0-15 age group (unlike kontanthjælp tables which start at 16). This is because some benefits relate to children (e.g. merudgiftsydelse for children with disabilities).
- All dimension columns have TOT rows. Filter alder='TOT', kon='TOT' for totals.
- Data ends at 2024-01-01.
