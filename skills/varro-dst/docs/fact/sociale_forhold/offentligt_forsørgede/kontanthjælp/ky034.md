table: fact.ky034
description: Kontanthjælp (personer og beløb) efter enhed, område og tid
measure: indhold (unit -)
columns:
- tal: values [1=Berørte personer (antal), 2=Helårsmodtagere (antal), 3=Udbetalt beløb (1.000 kr.)]
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- tid: date range 2007-01-01 to 2024-01-01
dim docs: /dim/nuts.md
notes:
- tal is a measurement selector with 3 values: 1=Berørte personer, 2=Helårsmodtagere, 3=Udbetalt beløb (1.000 kr.). Always filter to one value.
- omrade joins dim.nuts on omrade=kode. 98% match. omrade=0=Hele landet, omrade=997=Udlandet not in dim.nuts.
- Use ColumnValues("nuts", "titel", for_table="ky034") to see available regions. niveau 1=5 regioner, niveau 2=11 landsdele, niveau 3=98 kommuner.
- Data ends at 2024-01-01. For current regional data use ky04 (persons+payments) or ky16 (with ydelsestype).
- No ydelsestype or visitation breakdown — use ky035 for helårsmodtagere by visitation, ky038 for by origin.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
