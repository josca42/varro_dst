table: fact.ky038
description: Kontanthjælp (helårsmodtagere) efter område, ydelsestype, national oprindelse, herkomst og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode [approx]; levels [1, 2, 3]
- ydelsestype: values [10000=Ydelser i alt, 10050=Kontanthjælp i alt, 10025=Uddannelseshjælp i alt, 10060=Løntilskud vedrørende kontant- og uddannelseshjælpsmodtagere m. fl. I alt, 10070=Kontant- og uddannelseshjælp under forrevalidering i alt, 10020=Kontanthjælp og integrationsydelse til udlændinge m.fl. i alt, 10040=Revalideringsydelse i alt, 10032=Kontantydelse i alt (oktober 2015 -), 10035=Arbejdsmarkedsydelse i alt (januar 2014 -), 10030=Særlig uddannelsesydelse i alt ( -januar 2014)]
- herkomst1: values [TOT=I alt, 1=Personer med dansk oprindelse, 2=Vestlige lande, 9=Ikke vestlige lande, 000=Uoplyst national oprindelse]
- herkomst: join dim.herkomst on herkomst=kode [approx]
- tid: date range 2007-01-01 to 2024-01-01
dim docs: /dim/herkomst.md, /dim/nuts.md
notes:
- This table has NO tal selector — indhold directly represents helårsmodtagere only. No persons count or payments.
- omrade joins dim.nuts on omrade=kode. 98% match. omrade=0=Hele landet, omrade=997=Udlandet not in dim.nuts.
- herkomst column uses codes 0, 10, 20, 30, 40 — these do NOT match dim.herkomst (which uses 1, 2, 3, 9). Do NOT join to dim.herkomst. Treat as inline values: 0=I alt, 10=Dansk oprindelse, 20=Indvandrere, 30=Efterkommere, 40=Uoplyst.
- herkomst1 is national origin (inline): TOT, 1=Dansk, 2=Vestlige lande, 9=Ikke vestlige lande, 000=Uoplyst. herkomst and herkomst1 are nested — herkomst1 gives the finer country-group breakdown within each herkomst group.
- ydelsestype is hierarchical (10000=total). Filter accordingly.
- Data ends at 2024-01-01. Only table with geographic + ydelsestype + origin breakdown for helårsmodtagere.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
