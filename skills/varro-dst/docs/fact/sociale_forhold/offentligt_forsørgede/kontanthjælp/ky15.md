table: fact.ky15
description: Kontanthjælp (personer og beløb) efter enhed, ydelsestype (detaljeret) og tid
measure: indhold (unit -)
columns:
- tal: values [1=Berørte personer (antal), 3=Udbetalt beløb (1.000 kr.)]
- ydtypdetal: values [10000=Ydelser i alt, 10050=Kontanthjælp i alt, 573007=Kontanthjælp, 573012=Kontanthjælp til førtidspensionister uden ret til fuld social pension, 573019=Kontanthjælp til personer uden ret til social pension ... 579013=Kontantydelse, 579014=Løntilskud vedr. kontantydelse, 10035=Arbejdsmarkedsydelse i alt (januar 2014 -), 600001=Arbejdsmarkedsydelse i passive perioder, 600002=Arbejdsmarkedsydelse i aktiveringsperioder]
- tid: date range 2016-01-01 to 2025-06-01
notes:
- tal is a measurement selector: 1=Berørte personer (antal), 3=Udbetalt beløb (1.000 kr.). Always filter to one value.
- ydtypdetal has highly detailed ydelsestype codes (573xxx, 579xxx, 600xxx series) alongside aggregate codes (10000, 10050, etc.). Do not sum across all codes — the aggregates include the detailed rows.
- Only covers from 2016-01-01. For data before 2016 use ky12 or ky13 (with coarser ydelsestype).
- Use ColumnValues("ky15", "ydtypdetal") to see the full list of detailed payment type codes.
