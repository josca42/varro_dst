table: fact.ky16
description: Kontanthjælp (personer og beløb) efter område, enhed, ydelsestype og tid
measure: indhold (unit -)
columns:
- omrade: join dim.nuts on omrade=kode; levels [1, 2, 3]
- tal: values [1=Berørte personer (antal), 3=Udbetalt beløb (1.000 kr.)]
- ydelsestype: values [10000=Ydelser i alt, 10050=Kontanthjælp i alt, 10025=Uddannelseshjælp i alt, 10060=Løntilskud vedrørende kontant- og uddannelseshjælpsmodtagere m. fl. I alt, 10070=Kontant- og uddannelseshjælp under forrevalidering i alt, 10020=Kontanthjælp og integrationsydelse til udlændinge m.fl. i alt, 10040=Revalideringsydelse i alt, 10032=Kontantydelse i alt (oktober 2015 -), 10035=Arbejdsmarkedsydelse i alt (januar 2014 -), 10030=Særlig uddannelsesydelse i alt ( -januar 2014)]
- tid: date range 2007-01-01 to 2025-06-01
dim docs: /dim/nuts.md
notes:
- tal is a measurement selector: 1=Berørte personer (antal), 3=Udbetalt beløb (1.000 kr.). Always filter to one value.
- omrade joins dim.nuts on omrade=kode. 98% match. omrade=0=Hele landet and omrade=997=Udlandet are not in dim.nuts — exclude or handle separately when joining.
- Use ColumnValues("nuts", "titel", for_table="ky16") to see available regions. niveau 1=5 regioner, niveau 2=11 landsdele, niveau 3=98 kommuner.
- ydelsestype is hierarchical — 10000=Ydelser i alt covers all sub-categories. Do not sum across all ydelsestype codes.
- Combines regional and ydelsestype dimensions; for regional breakdown without ydelsestype detail use ky04.
- Map: /geo/kommuner.parquet (niveau 3), /geo/landsdele.parquet (niveau 2), or /geo/regioner.parquet (niveau 1) — merge on omrade=dim_kode. Exclude omrade=0.
